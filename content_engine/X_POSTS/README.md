# X_POSTS ― カテゴリ別X投稿完成文

最終更新: 2026-07-02 / 担当: コンテンツ班

`x_posts_by_category.csv` に、6カテゴリ・計22本のX投稿完成文を入れています（各140字目安・ハッシュタグ2〜3・記事1本へ送客）。

## 使い方
1. 投稿する行の `body` をコピー。
2. 末尾に `link` を貼る（Xは自動でカード表示。文字数に含めない運用でOK）。
3. `hashtags` を末尾に足す。
4. 手動投稿 → `POSTED_LOG.csv` に記録。
