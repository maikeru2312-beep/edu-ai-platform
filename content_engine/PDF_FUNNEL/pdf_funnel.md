（Desktop に保存済み。無料配布に使用）

## 目的とねらい
このサイトの目的は商品販売ではなく **PV・回遊・AdSense** です。PDFは「売るもの」ではなく、**サイトへ来てもらい・関連記事を回遊してもらうための入口**として使います。

- PDF単体をどこかに直置きするより、**関連記事ページ（school-generative-ai-privacy-security）に来てもらってから受け取ってもらう**形を基本にします（＝記事PVが立つ）。
- Xやnoteでは「無料で確認できるチェックリストがある」と伝え、記事URLへ誘導します。

## 導線の流れ
```
X / note の投稿
   │  「無料チェックリストがあります」
   ▼
記事ページ（school-generative-ai-privacy-security）
   │  記事内CTAでPDFを案内
   ▼
PDF受け取り（配布）
   │
   ▼
記事内の関連リンクで別記事へ回遊（PV増）
```

## 運用メモ
- PDFのダウンロード数は `KPI/kpi_content.csv` の `pdf_dl` 列に記録します。
- 反応が薄い場合は、告知文を「対象読者を名指しする」形（例：情報担当の先生／管理職）に変えて試します。
- 誇大な言葉（必読・完全・絶対など）は使いません。
