# 教育DXナビ Content Engine

最終更新: 2026-07-02 / 担当: コンテンツ班

---

## 1. このエンジンの目的

教育DXナビ（既存30記事・Next.js/AdSense収益サイト）への **検索外流入（X／note）を増やし、PV・回遊を上げる** ための、投稿素材の「作り置き」＋運用の仕組みです。

- 目的は商品販売ではなく、**PV／AdSense収益とサイト回遊**。
- 各投稿は既存記事1本へ送客します（`link` は `https://edu-ai-platform-delta.vercel.app/articles/<slug>`）。
- ユーザーは **毎回投稿文を考えません**。確認 → 軽微修正 → 手動投稿 → URL記録 だけで回せる状態にしてあります。

### やらないこと（制約）
- SNSの新規アカウントは増やしません（既存のX／noteのみ）。
- X・note APIや有料の予約投稿ツールに課金しません。
- すべて **手動投稿前提のコピペ素材** です。自動投稿はしません。

---

## 2. 運用モデル（役割分担）

| 担当 | やること |
|------|----------|
| コンテンツ班（このエンジン） | 投稿文・note下書き・スケジュール・テンプレの作成と補充 |
| **ユーザー** | ①内容の確認 ②必要なら軽微修正 ③手動でコピペ投稿 ④投稿URLを記録 |

ユーザーがやるのはこの4つだけです。文章をゼロから考える必要はありません。

---

## 3. 週次ワークフロー（15〜20分想定）

1. **月曜（準備・5分）**
   `POSTING_QUEUE.md`（週次カレンダー）で今週分を確認。気になる表現があれば `POSTING_QUEUE.csv` の `body` を直接修正。
2. **投稿日ごと（各1〜2分）**
   CSVの該当行 `body` ＋ `hashtags` ＋ `link` をコピーして、X／noteに手動投稿。投稿したら CSVの `status` を `posted` に更新。
3. **投稿後（各1分）**
   `POSTED_LOG.csv` に日付・プラットフォーム・投稿参照ID・実際のURL・反応数を追記。
4. **週末（ふりかえり・5分）**
   `KPI/kpi_content.csv` にその週の数値を入力し、`WEEKLY_REPORTS/TEMPLATE.md` をコピーして1枚書く。

> 数値は必ず実測値のみを入れます。推測やダミー値は入れません。

## 4. ファイル構成

| ファイル／フォルダ | 役割 |
|--------------------|------|
| `CONTENT_ENGINE.md` | このドキュメント（使い方の総覧） |
| `POSTING_QUEUE.csv` | 投稿スケジュール兼コピペ元。3〜4週間分をprefill。`body` はそのまま貼れる完成文 |
| `POSTING_QUEUE.md` | 上のCSVを人が見るための週次カレンダー表 |

| `X_POSTS/` | カテゴリ別のX投稿完成文（各140字目安・ハッシュタグ2〜3） |
| `FIXED_POSTS/` | 固定ポスト（ピン留め）案 |
| `PDF_FUNNEL/` | 無料PDF（リードマグネット）への導線設計と掲載文 |
| `KPI/kpi_content.csv` ＋ `KPI/README.md` | 投稿KPI記録表と列定義 |

| `WEEKLY_REPORTS/TEMPLATE.md` | 週次ふりかえり雛形 |
| `TEMPLATES/` | 穴埋め式の再利用テンプレ（X／note／スケジュール） |

## 5. 立場・トーン（全投稿共通の前提）

教員・学校現場の目線で書きます。

- 教育DXは、端末やアプリを増やすこと・子どもをデータ化して管理すること・教師の判断をAIに丸投げすることではありません。
- **子どもを見る／教材を整える／保護者と関わる時間を取り戻すための道具** として扱います。
- 特別支援は「便利さ」より **安心・見通し・個別性・安全性** を優先します。

### 文体ルール
- 煽らない／見下さない／断定しない／誇大表現を使わない。
- 公式情報は「こう読むと現場で使える」という **読み方** を添える。結論を押し付けない。
- 数値は正データのみ。勝手に作らない。

---

## 6. 送客対象カテゴリ（6）

特別支援教育／ICT活用／AI校務改善／教材・支援ツール／助成金・補助金／研修・セミナー。
`POSTING_QUEUE.csv` はこの6カテゴリをバランスよく回すよう設計しています。
