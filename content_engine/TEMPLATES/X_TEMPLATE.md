# X投稿テンプレート（穴埋め式）

最終更新: 2026-07-02 / 担当: コンテンツ班

140字目安・ハッシュタグ2〜3・記事1本へ送客。煽らない／断定しない／誇大表現なし。

---

## 基本型：現場の共感 → 具体の一手 → 記事へ

```
【共感】〈現場でよくある困りごと〉ですよね。
【視点】〈捉え直し／道具としての位置づけ〉。
【一手】まず〈すぐできる1つ〉から。
【誘導】〈テーマ〉の手順をまとめました。
〈記事URL〉
〈#ハッシュタグ2〜3〉
```

### 記入例（型の使い方の見本）
```
通知表の所見、白紙から書くと時間がかかりますよね。
AIは叩き台づくりに向いています。
まず伝えたいことを箇条書きにするところから。
所見をAIで下書きする手順をまとめました。
https://edu-ai-platform-delta.vercel.app/articles/chatgpt-tsuchihyo-shoken
#通知表 #教員のためのChatGPT
```

---

## 差し込み表
| 穴 | 入れるもの | 注意 |
|----|-----------|------|
| 〈困りごと〉 | 教員が実際に感じている負担 | 大げさにしない |
| 〈捉え直し〉 | 「〜を増やすことではなく、時間を取り戻す道具」 | 断定を避ける |
| 〈すぐできる1つ〉 | ハードルの低い行動 | 「まず1つ」に絞る |
| 〈記事URL〉 | `.../articles/<slug>` | 1投稿1記事 |
| 〈#タグ〉 | カテゴリ＋テーマの2〜3個 | 多用しない |

## チェック（投稿前）
- [ ] 140字前後に収まっているか
- [ ] 記事URLは1本だけか
- [ ] 誇大な語（必読・完全・絶対・激変 等）を使っていないか
- [ ] 公式情報は「読み方を添える」形になっているか
