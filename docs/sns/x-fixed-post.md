## 選定の考え方

3つの候補はそれぞれ異なる訴求軸を持っています。

- **候補1** は、はじめてプロフィールを訪れた人にサイトの全体像を素早く伝えたいときに向いています。アカウント開設直後や、フォロワーが少ない時期に最初に設定する固定ポストとして適しています。
- **候補2** は、発信への共感を入口にしたい場合に向いています。「教育DXって難しそう」「押しつけが苦手」と感じている層へのハードル低下を狙うため、フォロワーが増えてきた段階や、教育者コミュニティでの認知が広がってきた時期に切り替えると効果的です。
- **候補3** は、記事数が増えてきた時期や、サイトへのアクセスを増やしたいタイミングで有効です。「記事があること」を数字で示すことで、情報量への信頼感を伝えられます。

**ローテーションの目安**: 固定ポストを1〜3か月に1回程度見直し、フォロワー数・サイトの記事数・その時期の関心テーマに合わせて切り替えることを推奨します。

## 注意事項

- **記事数・カテゴリ数の更新**: 候補3に「25記事以上」と記載しています。記事を追加・削除した際は、実態に合わせて文言を更新してください。実態と異なる数字を掲載し続けると、読者の信頼を損なうおそれがあります。
- **URLの確認**: サイトのドメインや記事一覧のパスが変更された場合は、投稿内のURLを合わせて修正してください。
- **個人運営の明示**: いずれの候補も「個人運営であること」「所属機関を代表しない旨」を含めています。削除・省略しないよう注意してください。
- **文字数の余裕**: X（旧Twitter）の上限は日本語で140文字（全角）相当ですが、URLはt.co短縮後の23文字換算となります。URLを含む場合、本文＋23文字が140文字以内に収まるか、投稿前に実際の入力欄で確認してください。
- **ハッシュタグの追加は慎重に**: 固定ポストはプロフィールの「顔」です。過剰なハッシュタグはスパム的な印象を与えるため、追加する場合は1〜2個に留めることを推奨します。
