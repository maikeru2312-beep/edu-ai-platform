# 千冬先生＠教育DX 手動フォロー候補ガイド

**作成日**: 2026-06-19
**方針**: 手動・少量・投稿を読んでからフォロー

---

## 基本方針

- フォローはすべて**手動**で行う。ツール・スクリプト・自動化サービスは使用しない。
- フォローする前に、必ず**直近の投稿を数件読んで**から判断する。内容を確認せずにフォローしない。
- 量より質を優先する。多くフォローすることが目的ではなく、教育DX・特別支援教育に関心のある人とつながることが目的。
- **自動フォロー・一括フォローは行わない**。
- フォローバック目的（相互フォロー稼ぎ）でのフォローはしない。フォローされたからといって必ずフォロー返しする必要はない。

---

## 自動フォローをしない理由

- **X（Twitter）ポリシー違反リスク**: Xの利用規約では自動化・スパム的なフォロー行為を明確に禁止している。アカウントの凍結・機能制限につながる可能性がある。
- **低品質なつながりになる**: 投稿を読まずにフォローすると、教育と無関係なアカウントや商業目的のアカウントばかりになり、タイムラインの質が落ちる。
- **教育アカウントとしての信頼性へのダメージ**: 教員・教育関係者として発信しているアカウントが大量フォロー・スパム的行為を行うと、コミュニティ内での信頼が大きく損なわれる。
- **スパム認定リスク**: 短時間での大量フォローはXのシステムにスパムと判定されやすく、シャドウバンや警告の原因になる。

---

## 1日あたりのフォロー目安

- **目安：1日10〜20件以内**
- 新規アカウントや活動初期はさらに少なめ（1日5〜10件）に抑える。
- 1日の中で一度にまとめてフォローするのではなく、午前・午後など時間を分けて行う。
- 「今日は何件フォローした」と記録しておくと管理しやすい。
- フォロー数が増えてきたら（フォロー200件超えたあたりから）ペースを落とし、既存のつながりを深めることに注力する。

| `ICT活用` | 授業でのICT活用実践者 |

| `校務改善` | 校務DX・働き方改革を発信している教員・管理職 |
| `教員研修` | 研修・学び続ける教員のコミュニティ |
| `学校ICT` | 学校のICT環境・整備に関わるアカウント |

- 検索に出てきたツイートのリプライ欄にいる人もフォロー候補になることが多い。

## フォローしてよさそうなアカウント種別

### 教育ICTに関心がある教員
- 授業でのタブレット・PC活用を定期的に発信している。
- GIGAスクール端末の活用事例・実践記録を投稿している。
- 教員同士で情報共有・質問をしているアカウント。

### 特別支援教育に関心がある教員
- 特別支援学校・特別支援学級の担任・専科教員。
- 合理的配慮・インクルーシブ教育について発信している教員。
- AACや補助機器の活用を紹介している特別支援教育の実践者。

### 教育DXや生成AIについて発信している教育関係者
- ChatGPT・生成AIを授業や校務に取り入れた実践を発信している。
- 教育政策・文部科学省の動向を解説・コメントしている教育関係者。
- EdTechや教育テクノロジーの普及に取り組んでいる人。

### 教育系メディア・情報発信アカウント
- 教育ニュース・教育行政の情報を発信しているメディアアカウント。
- 教育委員会・教育センターの公式または非公式情報発信アカウント。
- 教員向けのオンラインコミュニティや勉強会のアカウント。

### 教材・ICT活用の情報発信アカウント
- 授業で使えるデジタル教材・ツールを紹介しているアカウント。
- プログラミング教育・STEM教育の実践者。
- 教員が自作したコンテンツやワークシートを公開しているアカウント。

## フォロー時のチェックポイント

フォローする前に以下を確認する。

- [ ] **直近の投稿を5件以上読んだか**: 内容が教育・ICT・特別支援教育に関連しているか確認する。
- [ ] **発信内容が本アカウントの方針に近いか**: 教育DX・GIGAスクール・特別支援教育のいずれかに関連していることが望ましい。
- [ ] **定期的に投稿しているか**: 直近1〜2ヶ月以内に投稿があることを確認する。数ヶ月以上更新がない休眠アカウントはフォローしない。
- [ ] **過度な商業目的・営業アカウントではないか**: 教育系ツール・サービスの宣伝ばかり投稿しているアカウントは慎重に判断する。
- [ ] **攻撃的・炎上目的の発信をしていないか**: 特定の人・組織への攻撃や不必要な論争を好むアカウントは避ける。
- [ ] **フォロワー数・フォロー数のバランスがおかしくないか**: フォロー数がフォロワー数の数十倍以上あるアカウントは相互フォロー目的の可能性がある。
- [ ] **プロフィールに教育・ICT関連の記載があるか**: 職業・活動内容が明記されているアカウントは信頼性が高い。

## フォロー後の交流方針

### いいね・返信・引用の使い分け
- **いいね**: 内容に共感・参考になったと感じたときに使う。数は気にしすぎない。
- **返信**: 質問・感想・追加情報を伝えるとき。短すぎる返信（「ですね！」のみなど）は避け、自分の意見や体験を一言添える。
- **引用**: 自分のフォロワーにも紹介したい・コメントを添えて広めたいときに使う。引用する際は相手への敬意を忘れない。

### 教員同士の交流マナー
- 意見が違っても批判ではなく「自分の現場ではこうです」という形で伝える。
- 相手の所属・立場を憶測で決めつけない。
- 返信が来なくても気にしない。全員が毎日Xを見ているわけではない。

### 特別支援教育コミュニティへの敬意
- 当事者・保護者・支援者が混在するコミュニティであることを意識する。
- 専門用語や支援技術の話題は、相手の立場を踏まえて慎重に発言する。
- 個人の事例・困りごとについての投稿にはセンシティブに対応する。

## やってはいけないこと

- **自動フォローツールの使用**: いかなるツール・スクリプト・サービスを使ったフォローも禁止。
- **一括フォロー（マスフォロー）**: 短時間で大量のアカウントをフォローしない。
- **フォロー稼ぎ（フォロバ目的のフォロー）**: 相手にフォロバさせることを目的としたフォローはしない。
- **フォロー→アンフォローの繰り返し**: フォローしてフォロバされたらすぐアンフォローする行為はXの信頼を損なう。
- **スパムリプライ・大量メンション**: 多数のアカウントに同じ内容のリプライを送ったり、無関係な話題でメンションしない。
- **プロフィールを確認しないフォロー**: 内容を読まずにキーワードだけで機械的にフォローしない。
- **非教育系・無関係アカウントへの大量フォロー**: 本アカウントのテーマと無関係なアカウントをフォローしてタイムラインを乱さない。

## 初回2週間のフォロー計画目安

アカウント開設直後・フォロー活動開始直後は特に慎重に行う。

### 第1週（1〜7日目）: 様子見・慎重スタート
- **1日あたりのフォロー目安: 5〜10件**
- まずは自分の発信（ツイート）を数件行い、アカウントとしての存在感を作る。
- 教育DX・GIGAスクール・特別支援教育のコアなキーワードで検索し、明らかに近い内容を発信しているアカウントを中心にフォロー。
- フォローした相手の投稿を読み、いいねや返信で自然に交流する。
- 1日の終わりに「今日フォローしたアカウントと理由」を簡単にメモしておくと振り返りやすい。

### 第2週（8〜14日目）: 少しペースアップ・質の維持
- **1日あたりのフォロー目安: 10〜15件**
- 第1週でフォローしたアカウントのフォロワー・フォロー中からさらに候補を探す（芋づる式）。
- ハッシュタグ検索やリプライ欄から新しいアカウントを発見する。
- フォローだけでなく、返信・引用で積極的に交流を始める。
- フォロー数が増えてきたら、タイムラインの質を確認し、関連性の低いアカウントはミュートやアンフォローも検討する。

### 2週間後の振り返り
- フォローしたアカウントのうち、実際に交流が生まれたものはいくつか確認する。
- タイムラインが教育DX・特別支援教育の情報で満たされているか確認する。
- 必要に応じてフォロー方針を見直す。

## 注意事項

- **Xポリシーの遵守**: Xの利用規約・コミュニティガイドラインは定期的に確認し、変更があった場合は対応する。スパム・自動化に関するポリシーは特に注意。
- **教員アカウントとしての信頼性維持**: 教育に関わる発信者として、フォロー行動も含めてオンライン上の行動に責任を持つ。炎上・トラブルになりそうな行動は避ける。
- **所属校・個人の特定につながる情報の管理**: フォローするアカウント・投稿内容から所属学校や勤務先が特定されないよう注意する。地域・学校名・クラス等が特定できる情報は投稿しない。
- **個人情報の取り扱い**: 児童・生徒・保護者に関わる情報はいかなる形でも投稿・コメントしない。
- **公私の区別**: 本アカウントは教育DX発信アカウントであることを意識し、私的な内容や政治・宗教等の発信は行わない。
- **体調・時間管理**: SNS活動に時間を使いすぎない。1日のフォロー・交流活動は15〜30分程度を目安にする。
