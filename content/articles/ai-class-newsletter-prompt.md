## はじめに

学年だよりや学級通信は、子どもたちの様子を保護者に伝える大切な文書です。一方で、毎月・毎週の発行に向けて文章を考える時間の確保が難しく、「ある程度の書き方でよいとわかっていても、言葉が出てこない」という場面は多いのではないでしょうか。

ChatGPTをはじめとする生成AIは、**文章のたたき台作成・箇条書きの文章化・表現の調整**に役立てることができます。ただし、学校現場では個人情報の取り扱いや内容の正確性への配慮が欠かせません。

本記事では、安全に使えるプロンプト例・Before/After形式の実践例・保護者に伝わる文章にするコツを、実際に活用できる形で紹介します。

> AIで作成した文書については、**内容の確認・修正・管理職への確認**を必ず行ってください。校内の文書作成・配布ルールに従って運用してください。

## 1. AIでできること・できないこと

まず、学級通信・学年だより作成においてAIが得意なこと・苦手なことを整理します。

### AIが得意なこと

- 箇条書きのメモを保護者向けの文章に整える
- 長い文章を短くまとめる
- 硬い表現をやわらかく言い換える
- 見出し・構成の案を出す
- 家庭へのお願い文の表現を整える
- 季節のあいさつ文を生成する
- 別のパターンの文章を複数出す

### AIには任せられないこと

- 実際の児童生徒の様子を判断する（AIは見ていない）
- 学校としての公式な立場・判断を代行する
- 個別の配慮・成績・家庭状況を扱う
- 事実確認なしに文章を完成させる
- 学校のルール・方針に沿っているか確認する

**基本の考え方：AIが下書きを作り、教員が仕上げる**

「AIが通信を作る」のではなく、「教員がAIの助けを借りて素早く仕上げる」です。最終的な内容の責任は、発行する教員・学校にあります。

## 2. 入力してよい情報・避ける情報

学校の文書作成にAIを使う前に、何を入力してよいかを確認してください。

### 入力してよい情報の例

- 一般的な行事内容・授業のテーマ（「生活科で春の植物を観察した」など）
- 全体としての児童生徒の様子（個人名なし）
- 家庭への一般的なお願い（「朝ごはんを食べて登校するよう声がけをお願いします」など）
- 架空の内容・一般的な例文
- 季節・時期に合わせたあいさつ文

### 入力してはいけない情報

- **児童生徒の氏名**
- **顔写真・個人が特定できる情報**
- **成績・評定・テストの点数**
- **障害名・診断名・要配慮情報**
- **家庭状況・保護者の個人情報**
- **生徒指導上の詳細・トラブルの経緯**
- **学校外秘の情報・未公表の方針**

入力する前に「この情報を外部サービスに送ってよいか」を確認する習慣をつけてください。

→ 個人情報とセキュリティの詳細は[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)をご参照ください。

## 3. AI活用の基本手順

学級通信・学年だよりをAIで作成する手順は次の5ステップです。

### Step 1：書きたい内容を箇条書きにする

まず、今回の通信に入れたい内容を自分でメモします。

```
・6月の行事（プール開始、学校公開日）
・今月の授業（国語：詩の音読、理科：昆虫の体の仕組み）
・子どもたちの様子（係活動が定着してきた）
・家庭へのお願い（水着の記名、熱中症対策）
```

### Step 2：個人名や詳しすぎる情報を確認・削除する

箇条書きの中に、児童名・特定の家庭のことが含まれていないか確認します。

### Step 3：ChatGPTに依頼する

目的・読者・文体・文字数を明記して依頼します（次のセクションで詳しく説明）。

### Step 4：出力された文章を確認・修正する

AIの出力を読み、**事実と異なる点・表現が合わない点・学校の文体と違う点**を修正します。

### Step 5：校内確認ルールに沿って配布する

管理職確認が必要な場合は確認を取り、印刷・配布の手順に従います。

## 4. すぐ使える基本プロンプト

以下のプロンプトは、いずれも**個人情報を含まない安全な例**です。`【　】`の部分を自分の内容に書き換えて使ってください。

### 学級通信の下書き（箇条書きから文章化）

```
以下の箇条書きをもとに、小学校の保護者向け学級通信の文章を作成してください。
個人名は使わず、クラス全体の取り組みや成長が伝わるようにしてください。
文体は丁寧で温かく、300字程度にしてください。

【内容】
・【行事・授業のテーマを書く】
・【子どもたちの様子を書く（個人名なし）】
・【次の活動の見通しを書く】
・【家庭へのお願いを書く】
```

### 学年だより（行事報告）

```
以下の内容をもとに、小学【○】年生の保護者向け学年だよりの行事報告文を作成してください。
個人名は使わず、学年全体の様子として書いてください。
丁寧な敬語で、400字以内にまとめてください。

【報告する行事名と日時】：
【当日の様子（箇条書き）】：
【子どもたちの感想や反応】：
【次の活動への見通し】：
```

### 保護者へのお願い文

```
以下の内容について、保護者向けのお願い文を作成してください。
命令口調にならず、協力をお願いする丁寧な表現にしてください。150字以内。

【お願いしたい内容】：
【理由・背景】：
【具体的にしてほしいこと】：
```

### 季節のあいさつ文

```
小学校の学級通信の冒頭に使う【○月】の季節のあいさつ文を作成してください。
子どもたちの学校生活に触れながら、保護者への感謝も入れてください。
丁寧な文体で150字以内。
```

### 授業の様子紹介

```
以下の授業の様子を、保護者向けに紹介する文章を作成してください。
個人名は使わず、クラス全体の取り組みとして表現してください。200字以内。

【教科・単元名】：
【主な活動内容】：
【子どもたちの様子・反応（個人名なし）】：
```

### 長い文章を短くする

```
以下の文章を、保護者が読みやすいよう200字以内にまとめてください。
重要な情報（日時・お願い事項）は残してください。

【元の文章を貼り付ける】
```

### やわらかい表現に直す

```
以下の文章を、保護者に親しみやすく温かい文体に書き直してください。
内容は変えずに、表現だけを調整してください。

【元の文章を貼り付ける】
```

## 5. Before / After 実践例

### 例①：箇条書きから学級通信の文章を作る

**【Before】教員の箇条書きメモ**

```
・生活科で学校の周りを探検した
・虫や植物をたくさん見つけた
・グループで話し合って記録カードにまとめた
・次は教室に戻って発表する
・外での活動の振り返りを家庭でもしてほしい
```

**【AIへの依頼文】**

```
以下の箇条書きをもとに、小学1年生の保護者向け学級通信の文章を作成してください。
個人名は使わず、クラス全体の様子が伝わるようにしてください。
丁寧で温かい文体で、300字程度にしてください。

・生活科で学校の周りを探検した
・虫や植物をたくさん見つけた
・グループで話し合って記録カードにまとめた
・次は教室に戻って発表する
・外での活動の振り返りを家庭でもしてほしい
```

**【After】AIが作った下書き（例）**

> 先日の生活科では、学校の周りを探検し、たくさんの発見をすることができました。草むらの中の小さな虫や、季節の植物を見つけた子どもたちは、目を輝かせながらグループで話し合い、記録カードにまとめることができました。
>
> 次回の授業では、見つけたことをクラスみんなに発表する予定です。もし機会がありましたら、「今日どんなものを見つけたの?」と、お子さんに問いかけてみてください。発見のよろこびをご家庭でも共有していただけると嬉しいです。

**【教員による修正のポイント】**

- 「目を輝かせながら」→ 実際にそういう様子だったか確認し、違う場合は削る
- 「次回の授業では」→ 正確な日程を追記する
- 「問いかけてみてください」→ 学校の文体に合わない場合は言い換える
- 全体の長さ・段落の区切り方を学校スタイルに合わせる

---

### 例②：事務的な文章を温かくする

**【Before】元の文章**

> 7月3日はプール開始日です。水着・水泳帽・タオル・ゴーグルを持参してください。健康観察カードに記入してください。

**【AIへの依頼文】**

```
以下の連絡文を、保護者に親しみやすく温かい文体に書き直してください。
内容（日時・持ち物）は変えずに、表現だけを調整してください。命令口調は避けてください。

7月3日はプール開始日です。水着・水泳帽・タオル・ゴーグルを持参してください。健康観察カードに記入してください。
```

**【After】AIが作った下書き（例）**

> いよいよ7月3日（水）からプール学習が始まります。お子さんと一緒に楽しみにしていただけると嬉しいです。
>
> 当日は水着・水泳帽・タオル・ゴーグルのご準備をお願いいたします。また、健康観察カードへのご記入もよろしくお願いいたします。安全に楽しくプールに取り組めるよう、体調を整えてご登校ください。

**【教員による修正のポイント】**

- 曜日を確認（7月3日が水曜か確認）
- 「楽しみにしていただけると嬉しいです」→ プール嫌いな子がいる場合は柔らかく
- 「ゴーグル」が必須かどうかを確認（任意の場合は表現を変える）

## 6. 用途別プロンプト集

追加で使いやすいパターンをまとめます。

### 行事の振り返り

```
【行事名】の振り返り文を保護者向けに書いてください。
個人名なし、クラス全体の取り組みと成長が伝わる内容で、300字以内。
【当日の内容や子どもたちの様子を箇条書きで書く】
```

### 長期休業前のお願い

```
夏休み前の学級通信に載せる、保護者へのお願い文を作成してください。
生活習慣・交通安全・インターネット利用の3点について、
家庭で意識してほしいことを丁寧にまとめてください。
全体で400字以内。
```

### 生活習慣のお願い

```
以下の生活習慣について、保護者への協力を依頼する文章を作成してください。
強すぎる表現にならず、お願いするトーンで、150字以内。
お願いしたい内容：【内容を書く】
```

### 学年全体の成長

```
【学期・月】の終わりに、子どもたちの成長を保護者に伝える文章を作成してください。
具体的なエピソードは入れず、全体の傾向として前向きにまとめてください。200字以内。
成長が見られた場面：【箇条書きで書く】
```

### 保護者への感謝

```
学期末の学級通信の締めに使う、保護者への感謝の言葉を書いてください。
形式的にならず、実感のある温かい表現で。100字以内。
```

### 短くしたいとき

```
以下の文章を半分の文字数に短くしてください。
重要な情報だけ残し、冗長な表現は省いてください。
【元の文章】
```

### 全体的に読みやすくしたいとき

```
以下の文章を、保護者が読みやすいように改善してください。
・段落を整える
・一文を短くする
・難しい言葉を平易にする
内容は変えないでください。
【元の文章】
```

## 7. 保護者に伝わる文章にするコツ

AIを使っても使わなくても、保護者向け文章で大切な4つの要素があります。

### ① 事実を書く

「何があったか」「何をしたか」を具体的に書きます。「楽しかった」だけでは伝わりません。「生活科の探検でダンゴムシを見つけ、図鑑で調べた」という事実が保護者の心に届きます。

### ② 成長を書く

「できたこと」だけでなく「どう取り組んだか」「前と何が変わったか」を伝えます。「静かに話を聞けるようになってきた」「友達の意見を聞いてから発言する場面が増えた」のように、**プロセスと変化**を伝えることが大切です。

### ③ 次の見通しを書く

「次は何をするのか」を伝えることで、保護者が家庭でも会話のきっかけを作りやすくなります。「次回は発表会です」「来週から○○の学習が始まります」という一行で、関心の継続につながります。

### ④ 家庭へのお願いは短く・明確に

「よろしくお願いします」だけでなく、「何を・いつ・どのように」を具体的に書きます。ただし、お願いが多すぎると読み飛ばされます。**1〜2点に絞って、明確に**伝えることを意識してください。

## 8. AIで作った文章をチェックする観点

AIの出力をそのまま使わず、次の観点で必ず確認・修正してください。

| 確認観点 | チェック内容 |
|---|---|
| **事実の正確性** | 実際にやっていない活動や、日時が違う内容が含まれていないか |
| **言い切りすぎ** | 「子どもたちは全員〜できました」など、実態と違う断定がないか |
| **個人の特定** | 「○○な子が」など、個人が推測できる表現がないか |
| **表現の温度感** | 大げさすぎ・事務的すぎ・命令的すぎていないか |
| **学校の文体** | 自校のスタイル・管理職の確認が必要な表現でないか |
| **保護者への誤解** | 意図しない受け取られ方をしないか |
| **著作権・引用** | AIが他の文章をそのまま出力していないか |

## 9. よくある失敗例

実際に起こりやすい失敗を事前に知っておくことで、対策できます。

**失敗① 個人情報を入れてしまう**

「A君が活躍しました」など、個人名を含むメモをそのままAIに貼り付けてしまうケースです。入力前に個人名・クラス・学校名が含まれていないか確認してください。

**失敗② AIの文章をそのまま使う**

AIが「全員が意欲的に取り組みました」と書いても、実際には乗り気でない子もいます。事実と異なる記述をそのまま配布することは、保護者の信頼を損ないます。必ず内容を確認・修正してから使います。

**失敗③ やっていない活動が文章に入る**

AIは入力された情報をもとに「もっともらしい文章」を生成します。「次回はまとめる予定」と書いたつもりが「まとめました」と書かれている、というミスが起こります。動詞の時制・活動の前後関係を必ず確認してください。

**失敗④ 保護者へのお願いが強すぎる**

「必ず○○してください」「〜しないでください」など、AIが強めの命令調で生成することがあります。お願い文は「〜ようご協力をお願いいたします」程度に調整してください。

**失敗⑤ 校内ルール確認をせずに使う**

ChatGPTなどの利用について学校・教育委員会のルールがある場合、確認なしに使い始めることはリスクになります。まず管理職・ICT担当に一言確認してから活用してください。

## まとめ

学年だより・学級通信へのAI活用は、**「丸投げ」ではなく「下書き・整理・言い換え」に使う**のが基本です。

**使い始めの手順：**

1. 書きたい内容を箇条書きにする
2. 個人名・詳細すぎる情報を除く
3. AIに目的・読者・文体・文字数を伝えて依頼する
4. 出力された文章を事実確認・表現修正する
5. 校内の確認ルールに沿って配布する

**守るべき3点：**

1. 児童生徒の個人情報・成績・家庭情報は入力しない
2. AIの出力はそのまま使わず、必ず確認・修正する
3. 学校・教育委員会のルールを確認してから使い始める

まずは「箇条書きのメモを保護者向けの文章に整える」使い方から試してみてください。慣れてくると、表現の引き出しが増え、通信作成にかける時間が自然と短くなっていきます。
