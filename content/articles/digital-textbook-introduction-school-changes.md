## はじめに

「来年度から、この学年はデジタル教科書が中心になるらしい」——そう聞いて、準備の見通しが立たないまま不安だけが先に立っている先生は少なくないはずです。端末は配られた、デジタル教科書のアカウントも届いた。けれど、紙をどうするのか、どの場面でどちらを使うのか、家庭学習はどう回すのか、そして特別支援が必要な子にとって本当に学びやすくなるのか——この肝心な「使い分けの設計」は、たいてい現場任せになっています。

さらに困るのは、「全部デジタルに切り替えれば効率的」という空気が、いつの間にか目的を追い越してしまうことです。紙を一斉になくした結果、これまで紙のページの位置で内容をつかんでいた子が、今どこを学んでいるか分からなくなって混乱する——こうした「便利にしたはずが、ある子にとっては見通しが崩れた」というつまずきは、導入初期に各地で繰り返されています。

GIGAスクール構想で1人1台端末が整備され、学習者用デジタル教科書の活用も段階的に広がっています。教科や学年を区切りながら本格導入が進められており、「これから紙の教科書はどうなるの?」「デジタルに切り替えたほうがよいの?」と気になっている先生も多いのではないでしょうか。

本記事は、特定の制度の是非を論じるものではありません。**「紙かデジタルか」という対立で考えるのではなく、どの子に・どの場面で・どちらが学びやすいかを考えるための視点**を整理することが目的です。

> **教育DXナビの立場**
>
> 教育DXナビでは、デジタル教科書を「紙を置き換えたかどうか」ではなく、「その子の読み・書き・見通しがどう変わったか」で考えます。デジタル化そのものはゴールではありません。子どもを見る時間・教材を整える時間を取り戻し、これまで紙では学びにくかった子に学びの入り口を増やすこと——それが達成されて初めて、導入に意味があると考えます。

> デジタル教科書の導入範囲・スケジュール・制度設計は段階的に進められており、変更される場合があります。最新の状況は、**文部科学省および所属校・教育委員会の案内を必ずご確認ください**。本記事は一般的な考え方の整理です。

## 1. そもそも「デジタル教科書」とは何か

「デジタル教科書」と一口に言っても、いくつかの種類があり、混同しやすいので最初に整理します。

| 種類 | 主な使い手 | 位置づけ |
|---|---|---|
| 学習者用デジタル教科書 | 児童生徒 | 紙の教科書と同じ内容を端末で読む教材 |
| 学習者用デジタル教材 | 児童生徒 | 動画・音声・ドリルなど、教科書を補う教材 |
| 指導者用デジタル教科書 | 教員 | 大型提示装置などで授業に提示するためのもの |

本記事で主に扱うのは、児童生徒が自分の端末で使う**学習者用デジタル教科書**です。ポイントは、デジタル教科書が「紙の置き換え」だけでなく、**音声読み上げ・拡大・書き込みなど、紙にはできない機能**を持っている点です。

→ 教育DXそのものの考え方は[教育DXとは何か？現場教員が押さえておきたい基礎知識](/articles/kyoiku-dx-kiso)もあわせてご覧ください。

## 2. 「紙かデジタルか」で考えない

デジタル教科書の話題になると、「紙のほうが頭に入る」「デジタルは便利」といった一般論の対立になりがちです。しかし現場で本当に大切なのは、**目の前の子ども・場面に合わせて選ぶ**という発想です。

紙とデジタルには、それぞれ得意なことがあります。

**紙が得意なこと**

- 一覧性が高く、全体を見渡しやすい
- 書き込み・付箋など手を動かす学びと相性がよい
- 端末の充電・操作トラブルに左右されない

**デジタルが得意なこと**

- 文字の拡大・読み上げ・色の調整ができる
- 書き込みのやり直し・共有がしやすい
- 動画・音声など多様な情報を扱える

つまり、「どちらが優れているか」ではなく、**「この場面では、どちらの強みが活きるか」**で考えるのが実態に合います。両方を場面で使い分けることも、当然の選択肢です。

判断に迷ったときの目安として、次の表を手元に置いておくと考えやすくなります。「どちらか一方に寄せる」ためではなく、**目の前の場面がどちらの列に近いかを見極める**ために使ってください。

| 場面 | デジタルが向きやすい | 紙が向きやすい |
|---|---|---|
| 読むことに困難がある子の学習 | 拡大・読み上げ・ルビで読みの負担を下げられる | — |
| 漢字や運筆など、手で書く力を育てる学習 | — | 鉛筆で書く経験そのものが学びになる |
| 全体を見渡して構造をつかむ場面 | 拡大すると全体が見えにくくなりやすい | 見開きで一覧でき、構造をつかみやすい |
| 動画・音声を伴う説明 | 教材内で再生でき情報が豊かになる | 扱えない |
| 通信・充電が不安定な家庭学習 | 端末トラブルに左右される | 確実に開ける |
| 見通しを大切にしたい子の学習 | 「今どこか」を補う工夫が要る | ページの位置で見通しを保ちやすい |

注意したいのは、**この表は「子どもごと」ではなく「場面ごと」の目安**だということです。同じ子でも、読むときはデジタル、書くときは紙、というように場面で切り替えてよい。安易に「この子はデジタルの子」と固定してしまうと、かえって選択肢を狭めます。

## 3. 使い分けを考える6つの観点

具体的に使い分けを判断するとき、次の観点で整理すると考えやすくなります。

### ① 発達段階

低学年では、指先の操作や読み書きの基礎を育てる段階でもあります。何でもデジタルにするのではなく、手で書く・紙をめくる経験とのバランスを意識します。

### ② 読みやすさ

文字を読むことに困難がある子にとって、拡大・読み上げ・行間調整ができるデジタルは大きな助けになります。一方で、画面の見え方が集中を妨げる子もいます。

### ③ 操作性

操作に気を取られて学習内容に集中できないなら、本末転倒です。「使えているか」より「使うことで学びやすくなっているか」を見ます。

### ④ 集中

通知や他機能に気が散りやすい子には、使う機能を絞る設定が役立ちます。紙のほうが集中できる場面もあります。

### ⑤ 書き込み

思考を書きながら整理したい場面では、紙やデジタルのペン機能のどちらが合うかを見ます。子どもによって相性が分かれます。

### ⑥ 家庭環境

家庭での通信環境・端末の持ち帰り可否によって、家庭学習での使い方は変わります。家庭の負担に配慮した運用が必要です。

これら6つは「どれか一つで決める」ものではなく、**重ね合わせて総合的に判断する**ものです。

## 4. 特別支援教育での視点

デジタル教科書は、特別支援教育の場面で特に大きな意味を持つことがあります。「便利な道具」という以上に、**学びへのアクセスを広げる手段**になり得るからです。

特別支援の観点で注目したい機能：

| 機能 | 支援につながる場面 |
|---|---|
| 文字の拡大・フォント変更 | 弱視・読みの困難がある子 |
| 音声読み上げ | 文字を読むことに時間がかかる子 |
| 背景色・文字色の変更 | まぶしさ・見えにくさがある子 |
| ハイライト・書き込み | 注目すべき箇所を絞りたい子 |
| ルビ表示 | 漢字の読みに支援が必要な子 |

一方で、注意したい点もあります。

- **操作の負荷**：機能が多いほど操作が複雑になり、かえって負担になることがあります。**今この子に必要な機能だけ**を有効にする絞り込みが大切です。
- **感覚過敏**：画面の明るさ・音・通知が刺激になる子には、明るさや音量、通知の調整を行います。
- **見通しのもちやすさ**：紙のほうがページの位置関係をつかみやすい子もいます。デジタルでは「今どこを学んでいるか」を補う工夫が要ります。見通しを支える具体的な方法は[特別支援教育における見通し支援：スケジュール・手順表・視覚支援の使い方](/articles/special-needs-visual-schedule-support)も参考になります。

「デジタルなら誰にでもよい」のではなく、**その子の実態に合わせて選び、調整する**という基本は、ICT活用全般と同じです。

→ 特別支援でのICT活用の全体像は[特別支援教育でICTを活用する：タブレット・AAC・読み上げソフトの選び方](/articles/tokubetsu-shien-ict)で詳しく扱っています。

## 5. 合理的配慮として位置づけるとき

デジタル教科書の機能を、特定の子どもの学びを支える手段として使う場合、それは**合理的配慮**の一環として整理できることがあります。

その際は、思いつきで導入するのではなく、次のプロセスを意識します。

1. 本人・保護者と必要性を話し合う
2. どの機能を・どの場面で使うかを整理する
3. 個別の教育支援計画・個別の指導計画に記載する
4. 実態の変化に合わせて見直す

記録に残しておくことで、**担任が変わっても支援が途切れにくくなります**。テストや検定での配慮の扱いは実施機関ごとに異なるため、個別の確認が必要です。

→ 合理的配慮の進め方と記録については[合理的配慮の申請と記録：学校現場での進め方ガイド](/articles/reasonable-accommodation-school-record)、計画書の書き方は[個別の教育支援計画・個別の指導計画の書き方](/articles/individual-education-plan-writing-guide)を参考にしてください。

## 6. 導入で起こりやすいつまずきと対応

デジタル教科書を使い始めるときに、現場でよく起こることと、その向き合い方を整理します。

| つまずき | 向き合い方 |
|---|---|
| 起動・ログインに時間がかかる | 授業前の準備手順を決め、子どもにも手順を共有する |
| どこを開いているか分からなくなる | ページ番号や見出しを声かけで共有する |
| 全員がデジタル一択になりがち | 紙と併用してよい場面を明示する |
| 家庭で開けない・通信が不安定 | 家庭学習はオフラインでできる課題も用意する |
| 操作に気を取られて内容が入らない | 最初は使う機能を絞り、慣れてから広げる |

大切なのは、**完璧を目指さず、小さく試して記録する**ことです。うまくいった工夫はメモして、学年や校内で共有すると、無理なく広がっていきます。

## 7. 現場でよくある失敗例

デジタル教科書の導入でつまずく原因の多くは、機能の善し悪しではなく、**「何のために・どの順番で使うか」の設計が抜けていたこと**にあります。匿名・一般化した形で、起こりやすい失敗を挙げます。

- **「効率化」のために全教科を一斉にデジタルへ切り替えた** … 紙のページ位置で内容を覚えていた子や、見通しを大切にする子が「今どこを学んでいるか」を見失い、混乱した。便利さを優先して、子どもの実態確認の順番を飛ばしてしまった例です。
- **デジタル教科書を「導入したかどうか」が目的化した** … 活用率や利用ログばかりが校内で話題になり、「その子の読みや理解がどう変わったか」を誰も見ていなかった。手段が目的を追い越した典型です。
- **読みに困難がある子にも、機能を全部オンにして渡した** … 読み上げ・ハイライト・ルビ・書き込みが一度に並び、操作に気を取られて肝心の内容が入らなくなった。「便利な機能が多い＝支援が手厚い」ではないという見落としです。
- **紙を一斉に回収してしまった** … 端末トラブルや通信不調の日に授業が止まり、「デジタルか紙か」ではなく「両方ないと回らない」現実が後から分かった。退路を残さず切り替えた順番の問題です。
- **保護者への説明を後回しにした** … 家庭学習で端末を開けない、目の負担が心配といった声が後から噴出し、信頼関係に影響した。導入前の説明という順番を抜かした例です。

いずれも、**目的と順番を先に設計していれば避けられた**つまずきです。「便利そうだから」「周りが始めたから」で進めると、こうした崩れ方をしやすくなります。

### 現場で考えるミニケース

ある学級で、文字を読むのに時間がかかる子がいました。担任は当初「この子はデジタルにすべきか、紙のままにすべきか」と悩んでいました。しかし発想を変え、「使うか使わないか」より先に**「何のために使うのか」**を考えました。たどり着いたのは、「音読の場面で、本人が内容を理解する時間を確保したい。だから読み上げ機能だけを使う」という目的です。機能は読み上げに絞り、書き込みやルビは当面オフにしました。結果として、その子は内容理解に集中でき、担任も「読めているか」を一人ひとり確認して回る時間を別の支援に回せました。**「デジタルにしたか」ではなく「何の時間を取り戻したいか」から決めた**ことが、うまくいった理由でした。

## 明日から確認できるチェックリスト

導入を進める前・進めた後に、立ち止まって確認したい観点です。すべてに「はい」と言える必要はありません。**「いいえ」が見つかったところが、見直しの出発点**になります。

- [ ] デジタル化が「便利そうだから」になっていないか（その子の学びや支援の質が変わるかで考えているか）
- [ ] 紙とデジタルを「場面で」使い分ける余地を残しているか（一斉に紙を回収していないか）
- [ ] 読みに困難がある子に、機能を全部オンにせず「今必要な機能だけ」に絞れているか
- [ ] 「今どこを学んでいるか」の見通しを保つ工夫があるか（見通しが崩れていないか）
- [ ] デジタル教科書を使うことで、教師が子どもを見る時間・支援に回す時間が実際に増えているか
- [ ] 個人情報やログの扱いが、校内ルール・教育委員会の方針と矛盾していないか
- [ ] 端末トラブル・通信不調の日でも授業や家庭学習が止まらない代替を用意しているか
- [ ] 家庭学習での使い方について、保護者へ事前に説明できているか

## 公式情報をどう読むか

デジタル教科書の導入範囲や制度設計は、**文部科学省**が公開する学習者用デジタル教科書に関する案内・実証事業の資料で確認できます。現場で読むときのポイントは、「いつから・どの教科で・どこまで紙と併用できるのか」という運用の幅を押さえることです。制度は段階的に変わるため、最終的には所属校・教育委員会から届く案内が優先します。

アクセシビリティ機能（読み上げ・拡大・色変更など）の具体的な仕様は教科書発行者やビューアごとに異なります。**どの機能が使えるか・どう設定するかは、導入されている製品のマニュアルや発行者の公式情報で確認**してください。「読み上げがあるはず」と思い込んで授業設計すると、当日に使えず崩れることがあります。

個人情報や学習ログの扱いについては、**個人情報保護委員会**や文部科学省が示す教育データの取り扱いに関する考え方が参考になります。「子どもをデータ化して管理すること」が目的化しないよう、何のためにログを見るのかを校内で確認しておくと安心です。いずれの情報も変更される場合があるため、利用前に各機関の公式情報で最新の内容をご確認ください。

## まとめ

デジタル教科書の本格導入が段階的に進むなかで、現場が押さえておきたい考え方を整理します。

**基本の姿勢**
- 「紙かデジタルか」の二択で考えない
- どの子に・どの場面で・どちらが学びやすいかで選ぶ
- 両方を場面で使い分けてよい

**使い分けの観点**
- 発達段階・読みやすさ・操作性・集中・書き込み・家庭環境を重ねて判断する

**特別支援の視点**
- 拡大・読み上げ・色の調整は学びへのアクセスを広げる
- 操作負荷・感覚過敏・見通しに配慮し、必要な機能だけ絞る
- 合理的配慮として、計画書に記載し見直す

デジタル教科書は「紙を置き換えるもの」ではなく、**子ども一人ひとりに合った学び方の選択肢を増やすもの**です。最新の制度は公式情報で確認しつつ、現場では小さく試しながら、自校に合った使い方を探していきましょう。

参考情報：文部科学省「学習者用デジタル教科書」に関する案内（最新の導入状況は文部科学省の公式サイトでご確認ください）
