## はじめに

特別支援教育の現場には、さまざまなICT支援ツールが登場しています。見通しを示すタイマーアプリ、文字を音声で読み上げるツール、コミュニケーションを助けるAACアプリ——それ自体は使いやすくできていても、「どれを選べばよいのか」「子どもに合っているのか」という判断に迷う教員は少なくありません。

ICT支援ツールを選ぶ基準は、「便利そうかどうか」ではありません。**子どもが何に困っているのか、何ができるようになりたいのか**、その目的から逆算して選ぶことが出発点です。

本記事では、ICT支援ツールを目的別に整理し、授業での使い方と導入前に確認しておきたいことをまとめます。

> 本記事は一般的な情報提供を目的としています。具体的な支援方針・ツール導入の判断は、本人・保護者・関係機関と連携の上、**所属校・教育委員会の方針に従って**進めてください。個別の医療的・診断的判断を代行するものではありません。

## ICT支援ツールを選ぶ前に確認したいこと

ツールを探し始める前に、次の問いを整理しておくと選択の軸がぶれにくくなります。

### 子どもが何に困っているのか

「使えそうなツール」より先に、「今、何が難しくて、授業への参加が制限されているのか」を具体的に把握することが最初の一歩です。

困りごとの例：

- 言葉で意思を伝えることが難しい
- 文字を読むことに時間がかかる
- 授業の流れが読めず、次に何をすればよいかわからなくなる
- 手書きで文字を書くことに強いストレスがある
- 感覚の調整が難しく、集中を保てない時間がある

### 何をできるようにしたいのか

「この子をICTで支援したい」という気持ちだけでは、目的が曖昧になります。**「授業への参加」「自分の考えの表現」「理解の確認」「見通しを持った行動」「他者とのコミュニケーション」**など、目的を分けて考えるとツールの候補が絞りやすくなります。

## 目的別のICT支援ツールの考え方

### 1. 見通し支援

「次に何があるのか」「あとどれくらいで終わるのか」が分かることは、見通しを持つことが難しい子どもにとって大きな安心につながります。

考え方：

- タイマー表示や視覚的なカウントダウンは、「時間の流れ」を見える形にする
- 1日の流れや授業の手順を写真・アイコン・文字で示すスケジュール表示は、行動の見通しを支える
- 「終わった活動」を消す・裏返すといった操作が自分でできると、達成感にもつながる

注意点：ツールを使うことでかえって画面に注意が向きすぎることもあります。紙のスケジュール表や具体物と組み合わせて使う方法も有効です。

→ 視覚スケジュールの活用については[特別支援教育での視覚スケジュール・構造化支援の実践](/articles/special-needs-visual-schedule-support)もご参照ください。

### 2. コミュニケーション支援（AAC）

言葉による表現が難しい子どもが、自分の意思・気持ち・要求を伝えるための手段を広げることがAACの目的です。ICTはその選択肢の一つです。

考え方：

- 写真・絵カード・シンボルをタップして意思を伝えるアプリは、言語表現の代替手段として使われる
- 音声出力機能があるツールは、声で伝えることが難しい場面で活用できる
- 子どもが自分で選んで操作できる画面設計か、スイッチアクセスなど操作方法の選択肢があるかを確認する

注意点：AACはあくまで「伝える手段の選択肢を広げる」ためのものです。「言葉を使わなくなる」という懸念はよく挙がりますが、AACを使うことで言語表現が抑制されるという根拠はありません。子どもの実態に合わせて、言語指導や他の支援と組み合わせて使います。

→ ICTとコミュニケーション支援の全体的な考え方については[特別支援教育でICTを活用する](/articles/tokubetsu-shien-ict)もご参照ください。

### 3. 読み書き支援

文字の読み書きに困難がある子どもに対しては、ICTが「別の方法で読む・書く」手段を提供できます。

考え方：

- テキスト読み上げ機能は、文字を視覚的に読むことが難しい場合に音声で情報を受け取る手段になる
- 音声入力・スキャンなどの入力補助は、手書きが難しい場合に「書く」手段を広げる
- ふりがな表示、フォントサイズ変更、文字間隔の調整など、読みやすさに関わる設定が変更できるかを確認する
- 文字変換候補の表示など、スペルや文字の想起が難しい場合を支える入力支援機能もある

注意点：読み書き支援ツールは「読み書きの練習をしなくてよい」ということではありません。何の目的で使うかを明確にし、他の学習活動と切り分けて使います。

### 4. 記録・振り返り支援

学習記録や自己評価を積み重ねることが難しい場面でも、ICTが記録の負担を下げる役割を果たせます。

考え方：

- 写真・動画・音声での記録は、文字での記録が難しい子どもが「やったこと・できたこと」を残す方法になる
- ポートフォリオ的に記録を蓄積することで、本人・保護者・関係教員が変化を確認しやすくなる
- 日々の気持ちや体調の記録（体調チェック・感情のセルフモニタリング）を視覚的に入力できるツールもある

注意点：記録に使った写真や音声には個人情報が含まれます。保存先・共有範囲・管理方法について、校内ルールと保護者への説明を事前に整理しておく必要があります。

→ 個人情報の取り扱いについては[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)も参考にしてください。

### 5. 感覚・注意の調整支援

感覚の過敏さや注意の持続に困難がある場合、ICTは環境調整の一部として機能することがあります。

考え方：

- ノイズキャンセリング機能付きヘッドホンやイヤーマフは、聴覚的な刺激を調整する補助具として使われる
- 集中しやすい画面表示（余計な情報を減らす、背景色を変える）を設定できるツールもある
- 一定時間ごとに休憩を促す機能や、視覚的なポーズのタイミング提示を活用する方法もある

注意点：感覚の特性は子どもによって大きく異なります。「このツールを使えばよい」という決まった答えはなく、個別の実態を確認しながら試行・調整していくことが基本です。医療的な判断が必要な場面では、専門機関と連携して進めます。

### 6. 教員の教材準備支援

ICT支援ツールは、子どもが使うだけでなく、教員が教材を準備したり支援計画を整理したりする場面でも活用できます。

考え方：

- 個別の指導計画（IEP）の文書作成を効率化する書式ツールやテンプレートの活用
- 視覚支援教材（スケジュール表・絵カード・課題指示）のデジタル作成・印刷
- 校内での情報共有や引き継ぎをシンプルにまとめる記録ツールの活用

注意点：教材準備でAI生成ツールを使う場合、児童生徒の個人情報（名前・成績・特性）を入力しないことが原則です。

→ 生成AIを校務で安全に使う観点は[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)をご覧ください。

## 目的別ICT支援ツールの選び方（整理表）

| 目的 | 主な活用場面 | 確認したい観点 |
|---|---|---|
| 見通し支援 | タイマー・スケジュール表示 | 子どもが自分で操作できるか、終了が分かりやすいか |
| コミュニケーション支援 | 意思表示・要求伝達 | 語彙・操作方法が子どもの実態に合っているか |
| 読み書き支援 | 読み上げ・音声入力・文字調整 | 授業場面・目的に合った機能があるか |
| 記録・振り返り支援 | 写真・音声記録・気持ちの記録 | 個人情報の保存・管理方法が明確か |
| 感覚・注意の調整 | 環境調整・休憩タイミング提示 | 個別の感覚特性に合っているか、過剰な刺激を与えないか |
| 教材準備支援 | 指導計画・視覚教材作成 | 個人情報を入力しない運用ができるか |

## 授業で使うときの注意点

### 使う時間・場面を決める

「授業全体でずっと使う」のではなく、「この活動のこの部分で使う」と場面を絞る方が、子どもも教員も混乱しにくくなります。いつ使い、いつしまうかを事前に決めておきましょう。

### 子どもが自分で選べる余地を残す

支援ツールを「使わせる」のではなく、子ども自身が「使うかどうか」「どう使うか」を選べる環境が理想です。初めから全員・全場面で使わせるより、「必要な時に自分で取り出す」運用が子どもの主体性を育てます。

### 紙や具体物と組み合わせる

ICTはあくまで選択肢の一つです。紙のスケジュール表、絵カード、実物、板書など、ICT以外の方法とうまく組み合わせることで、子どもが様々な場面で対応できる力がつきます。ICT依存を避けるためにも、デジタルとアナログを意識的に使い分けましょう。

### 操作練習だけで終わらせない

「アプリを操作できるようになること」が目標ではありません。ツールを使って何ができるようになるかが学習のゴールです。操作に慣れた後は、ツールを使って本来の学習活動に向かっているかを確認します。

### 個人情報・写真の管理に注意する

記録に使った写真・動画・音声には個人情報が含まれます。端末の管理場所、外部への送信・共有、データ削除のタイミング、保護者への説明など、学校のルールと照らし合わせて運用してください。

→ GIGAスクール端末の管理・運用に関する観点は[GIGAスクール端末を授業で活かす基本](/articles/giga-device-lesson-use-guide)もご参照ください。

## 授業での注意点まとめ

| 注意点 | 具体的に確認すること |
|---|---|
| 使う場面を絞る | いつ使い始め、いつしまうかを事前に設定しているか |
| 子どもの選択を大切に | 子どもが「使うかどうか」を自分で判断できる運用になっているか |
| 紙・具体物と組み合わせる | ICTだけに依存せず、他の方法も選べるようにしているか |
| 目的を忘れない | 操作できるようになることではなく、学習参加が広がっているかを確認しているか |
| 個人情報の管理 | 写真・音声記録の保存・共有・削除のルールが整理されているか |

## 支援ツール選定チェックリスト

ICT支援ツールを導入・継続する前に、以下の観点を確認しておくと、導入後のトラブルや使い続けられない事態を防ぎやすくなります。

### 目的の明確さ

- [ ] 「何に困っているか」を具体的に把握しているか
- [ ] 「何ができるようになりたいか」が明確か
- [ ] ツールを使うことが目標になっていないか

### 子どもとのマッチング

- [ ] 子ども自身が操作できるか（または支援があれば使えるか）
- [ ] 子どもがそのツールを使うことに抵抗感はないか
- [ ] 子どもの発達段階・感覚特性に合っているか

### 継続・引き継ぎ可能性

- [ ] 担当教員が準備・更新を継続できるか
- [ ] 学年をまたいで引き継ぎができる仕様か
- [ ] 家庭でも共有・連携できる必要があるか、保護者への説明をしているか

### 安全・管理

- [ ] 個人情報（写真・音声・記録）の保存先・管理方法が明確か
- [ ] 外部サービスへのデータ送信について学校のルールを確認しているか
- [ ] 導入・更新に際して管理職・情報担当と確認済みか

| 観点 | 確認事項 |
|---|---|
| 目的 | 子どもの困りごとと使用目的が対応しているか |
| 操作性 | 子ども本人がある程度自分で使えるか |
| 継続性 | 教員が準備・更新を継続できる負担か |
| 引き継ぎ | 次年度・他教員に引き継げる運用か |

| 確認状況 | 管理職・情報担当への報告・確認が済んでいるか |

## まとめ

ICT支援ツールは、子どもの学びへの参加や表現の可能性を広げるために使います。「このツールを使えば解決する」という発想ではなく、「この子のこの困りごとに対して、この目的でこう使う」という具体的な場面設定が出発点です。

最初から完璧な運用を目指す必要はありません。**まずは一つの困りごとに対して、小さく試す**ことが継続のコツです。試してみて合わなければ別の方法を考え、合っていれば少しずつ場面を広げていきましょう。

ICT支援ツールの選択と活用は、子どもの実態を一番よく知っている教員と、保護者・関係機関が連携して判断するものです。特定の製品を導入することが目標ではなく、その子が「参加できている」「伝えられている」「分かっている」という状態を積み重ねることが本来のゴールです。
