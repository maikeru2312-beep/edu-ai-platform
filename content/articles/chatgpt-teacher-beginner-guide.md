## はじめに

「ChatGPTという言葉は聞いたことがあるが、何ができるのかよくわからない」「学校で使っても問題ないのか不安」「どこから始めればよいかわからない」——そんな教員の方に向けた入門記事です。

ChatGPTは教員の仕事にも活用できる道具ですが、**学校現場では「便利さ」と「安全性」をセットで考えることが欠かせません**。本記事では、アカウント作成から基本操作・校務での使い方・注意すべき点まで、ステップごとに整理します。

> ChatGPTの利用については、**所属校・教育委員会のガイドライン・方針を必ず事前に確認**してください。自治体や学校によって利用可能なサービス・範囲が異なります。

## 1. ChatGPTとは何か

ChatGPT（チャットジーピーティー）は、米国のOpenAI社が開発した**対話型の生成AIサービス**です。テキストを入力すると、AI が文章・リスト・表・アイデアなどを生成して返してくれます。

### 検索エンジンとの違い

| | 検索エンジン | ChatGPT |
|---|---|---|
| **使い方** | キーワードを入れてページを探す | 会話するように依頼する |
| **返ってくるもの** | 既存のWebページの一覧 | AIが生成した文章・案・整理 |
| **強み** | 最新情報・信頼できるソース | 文章作成・整理・言い換え・アイデア出し |
| **弱み** | 文章を作ってはくれない | 誤りを含む場合がある・最新情報は苦手 |

「何かを調べる」よりも「何かを書く・整理する・考える入口を作る」のが ChatGPT の得意なことです。

### 重要：ChatGPTは間違える

ChatGPT はもっともらしい文章を生成しますが、**事実と異なる内容（ハルシネーション）を自信満々に出力することがあります**。生成された内容は必ず教員が確認・修正してから使用してください。医療・法律・成績判断など、専門的な判断が必要な事柄をAIに任せることは避けてください。

## 2. 教員がChatGPTでできること

以下は、個人情報を含まない範囲で教員が活用しやすい場面の例です。

| 業務 | 活用内容 |
|---|---|
| 学級通信・おたより | 文章のたたき台・表現の見直し |
| 授業準備 | 導入アイデア・学習活動の案出し |
| 教材文の言い換え | 難しい説明を児童向けに平易化 |
| 所見・記録文 | 表現のバリエーション出し（個人情報は入れない） |
| 会議・研修 | 配付資料の構成案・議事メモの要点整理 |
| 保護者向け文書 | 文体・表現の調整 |
| 校務チェックリスト | 業務手順や確認事項のリスト化 |

いずれも「AIが作った文章をそのまま使う」のではなく、**「AIの下書きを教員が確認・修正して仕上げる」という使い方が基本**です。

## 3. まず何から始めるか

### ① 所属校のルールを確認する

使い始める前に、**ICT担当・管理職に一言確認**してください。「ChatGPTを校務の文章作成に試したいのですが、問題ありませんか?」という確認だけで十分です。

### ② アカウントを作成する

ChatGPT は [openai.com](https://openai.com) からアカウントを作成して利用できます。

- **無料プランあり**（機能に制限あり）
- **有料プラン（ChatGPT Plus）** — より高性能なモデルを利用可能、月額費用あり
- 学校・自治体が法人契約しているサービスがある場合は、そちらを優先して使う

アカウント作成には**メールアドレス**が必要です。学校配布のメールアドレスを使うか個人アドレスを使うかは、校内ルールを確認してください。

### ③ 最初は架空・一般的な内容で試す

いきなり校務データを入力せず、まず**個人情報を含まない架空の内容や一般的な文章**で試してみましょう。

**試しやすい最初の依頼例：**
- 「6月の学級通信のリード文を200字で書いてください」
- 「小学4年生向けに、水の循環を簡単に説明してください」
- 「保護者会の司会進行の冒頭あいさつ文を作ってください」

操作そのものは非常にシンプルです。入力ボックスに依頼を書いて送信するだけです。

## 4. 基本的な使い方

### 具体的に依頼する

曖昧な依頼では曖昧な結果が返ります。**対象・目的・文字数・文体**を指定すると精度が上がります。

| 指定する要素 | 例 |
|---|---|
| 対象 | 「小学3年生の保護者向けに」 |
| 目的 | 「運動会のお知らせ文として」 |
| 文字数 | 「300字以内で」 |
| 文体 | 「丁寧な敬語で」「箇条書きで」 |

### 出力された文章をそのまま使わない

AIの出力は**必ず読んで確認し、事実と異なる点・表現が合わない点を修正**してから使います。

### 追加で修正依頼できる

一度の生成で満足できなければ、続けて修正を依頼できます。

- 「もう少し短くしてください」
- 「保護者が読みやすいように、やさしい表現に変えてください」
- 「箇条書きにしてください」
- 「もう少し温かみのある文体にしてください」

対話を重ねながら仕上げていくのが、ChatGPT の使い方の基本です。

## 5. 教員向けプロンプト例

以下はすべて、**個人情報を含まない安全な入力例**です。

### 学級通信のたたき台

```
6月の学年だより（小学2年生の保護者向け）の本文を作成してください。
テーマ：梅雨の季節の過ごし方と、7月の行事のお知らせ
文字数：400字程度、丁寧な敬語
```

### 授業の導入アイデア

```
小学5年生の理科「植物の発芽と成長」の単元で、
子どもたちの興味を引く授業の導入アイデアを3つ提案してください。
実験や観察を取り入れた内容が望ましいです。
```

### 難しい文章を言い換える

```
以下の文章を、小学3年生が理解できるようにやさしく言い換えてください。

（元の文章を貼り付ける）
```

### 会議メモの要点整理

```
以下の会議メモから、決定事項・確認事項・次回への持ち越し事項を
整理してまとめてください。

（個人名・内部情報を除いた箇条書きメモを貼り付ける）
```

### 支援の観点を整理する（個人情報なし）

```
授業中に集中を保つことが難しい小学生への支援として、
教員が工夫できる環境整備・声かけ・教材の工夫を10個リストアップしてください。
特定の児童の情報ではなく、一般的な支援策として提案してください。
```

## 6. 学校現場で使うときの注意点

### 入力してはいけない情報

以下の情報は、ChatGPT をはじめとする外部サービスに入力しないでください。

- **児童生徒の氏名・学籍番号**
- **成績・評定・テストの点数**
- **障害名・診断名・家庭状況・要保護情報**
- **住所・電話番号などの連絡先**
- **写真・動画・音声データ**
- **学校内部の未公開情報**（人事・予算・未決定の方針など）

### なぜ問題になるか

ChatGPT に入力した内容は、OpenAI のサーバーに送信されます。利用規約によっては学習データとして使われる可能性があり、**個人情報保護法・校内規定との兼ね合いが生じます**。

「無料だから問題ない」「匿名にしたつもり」という判断ではなく、入力前に「これを外部サービスに送ってよいか」を確認する習慣を持ってください。

### 出力内容をうのみにしない

ChatGPT の出力は**もっともらしいが誤りを含む場合があります**。特に次の場面では必ず事実確認を行ってください。

- 法令・通知・文部科学省のガイドラインに関する記述
- 具体的な数値・日付・統計データ
- 医療・福祉・心理に関する情報

### 著作権に注意する

ChatGPT が生成した文章の著作権については現在も議論が続いています。生成した文章を公表・配布する場合は、内容を教員が十分に手を加えて「自分の著作物」として仕上げることが現実的な対応です。

→ 個人情報・セキュリティの詳細は[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)をご参照ください。

## 7. 校務で使う場合の線引き

| 区分 | 例 | 目安 |
|---|---|---|
| **比較的始めやすい** | 全体向け文書・通知文・アイデア出し・一般的な教材文 | 個人情報を含まない文章作成 |
| **慎重に判断する** | 特定の子どもの支援方針、保護者との個別連絡 | 内容を十分確認・修正した上で活用 |
| **使わない方がよい** | 成績・評定の決定、指導要録の記載、家庭情報を含む文書 | 専門的判断・個人情報が必須の業務 |

校務利用では、**学校・自治体が許可した環境・サービスを使う**必要がある場合があります。個人契約のサービスで校務データを扱うことを禁止している学校も多いため、ルールの確認が前提です。

→ 校務でのAI活用全般については[AIで変わる校務改善：教員の残業を減らす5つの活用法](/articles/ai-koomu-kaizen-nyumon)もご参照ください。

## 8. ChatGPTを授業で使う場合

### 教員が教材準備に使う場合

本記事の内容で対応できます。個人情報を含まない範囲で、教材案・説明文の言い換え・発問リストの作成などに活用できます。

### 児童生徒が直接使う場合

児童生徒が ChatGPT を直接使う授業を行う場合は、以下の確認が必要です。

- 学校・教育委員会の利用許可があるか
- 年齢制限（ChatGPT の利用規約では13歳以上、保護者同意が必要な年齢もあり）
- アカウントをどのように管理するか
- 情報モラル・個人情報の指導を事前に行っているか
- 生成AIを使ったことを踏まえた評価の考え方

生成AIは「調べれば答えが出る」体験を提供しますが、**子どもが考える・試行錯誤する経験を置き換えるものではありません**。「AIが出した答えを批判的に読む」「AIに頼らずに考える場面を作る」という授業設計のバランスが重要です。

→ 授業でのICT活用全般については[タブレット1台で始める授業ICT活用【GIGAスクール対応】](/articles/tablet-ict-jugyo-giga)もご参照ください。

## 9. よくある不安とQ&A

**Q. 学校でChatGPTを使ってよいのか?**

A. 学校・教育委員会によって方針が異なります。使い始める前に必ず管理職またはICT担当に確認してください。「禁止されていない=自由に使える」とは限りません。

**Q. 無料版でも使えるのか?**

A. 無料版でも基本的な文章作成・整理・アイデア出しは十分できます。高度な機能（画像生成・より高精度なモデル）は有料プランが必要ですが、初心者には無料版から試すことをお勧めします。

**Q. 個人情報を入れなければ大丈夫なのか?**

A. 個人情報を入れないことは最低限の条件ですが、それだけで「大丈夫」とは言い切れません。学校・自治体のルールに従うことが前提です。

**Q. 出てきた文章をそのまま使ってよいのか?**

A. そのまま使うことはお勧めしません。事実確認・表現の修正・学校の文体への調整を行ってから使用してください。

**Q. AIが間違えることはあるのか?**

A. あります。ChatGPT はもっともらしい文章を生成しますが、事実とは異なる内容（ハルシネーション）を生成することがあります。特に統計・法令・日付・固有名詞は必ず確認してください。

**Q. 保護者向け文書に使ってよいのか?**

A. 全体向けの通知文・お便りのたたき台として活用することは、個人情報を含まない範囲であれば比較的導入しやすい使い方です。ただし内容の正確性確認・学校の文体への調整・最終確認は教員が必ず行ってください。

## まとめ

ChatGPT は教員の仕事を置き換えるものではなく、**下書き・整理・発想の入口を助ける道具**です。

**使い始めの3ステップ：**

1. **校内ルールを確認する** — ICT担当・管理職に「使って問題ないか」を確認
2. **個人情報を含まない内容から試す** — 学級通信・教材案・アイデア出しから始める
3. **出力は必ず確認・修正する** — AIの下書きを教員が仕上げて使う

**守るべき3原則：**

1. 児童生徒の氏名・成績・診断名・家庭情報は入力しない
2. 生成された文章はそのまま使わず、必ず確認・修正する
3. 「便利さ」と「安全性」をセットで判断する

「まず試してみる」前に「ルールを確認する」——このひと手間が、安心して使い続けるための基本です。小さな一歩から始めましょう。
