## はじめに

GIGAスクール構想により、全国の学校に1人1台端末が整備されました。しかし「端末はあるが、毎時間どう使えばよいかわからない」「ICTが苦手で授業に取り入れることへのハードルが高い」という声は、多くの教員から今も聞かれます。

ICT活用は「特別な授業」を作るためのものではありません。**いつもの授業を、もう少し見やすく・考えやすく・共有しやすくするための手段**です。

本記事では、タブレット1台から無理なく始められる授業ICT活用を、場面別・教科別・注意点まで教員向けに整理します。

> 端末の種類・校内ネットワーク・使用できるアプリは学校・教育委員会によって異なります。具体的な活用内容については**所属校のICT担当・管理職に確認の上**進めてください。

## 1. 授業ICT活用で大切な考え方

### 端末を使うことが目的ではない

「今日はタブレットを使った」ことが成功ではなく、「子どもたちが考えを深めた」「表現しやすくなった」「理解が広がった」ことが目的です。ICTはそのための道具のひとつです。

使わない方が良い場面もあります。紙の方が速い・手を動かす経験が必要・対話の流れを止めたくない——こうした判断も、ICT活用の一部です。

### 6つの目的で整理する

ICTを授業で使う目的は、大きく次の6つに整理できます。

| 目的 | 具体的なイメージ |
|---|---|
| **見る** | 写真・動画・資料を大きく・鮮明に提示する |
| **調べる** | Webや教材データベースで情報を探す |
| **考える** | メモ・図・マップで思考を整理する |
| **表す** | 文章・写真・スライドで表現する |
| **共有する** | クラス全体やグループで考えを見せ合う |
| **振り返る** | 学習の記録を残し、成長を確認する |

授業を設計するときに「この時間はどれを目的にするか」を決めると、ICTの使い方が自然に決まります。

### まずは教師用1台から始める

子ども全員の端末を使う前に、**教師用端末1台を大型提示装置（テレビ・プロジェクタ）につなぐ使い方**から始めると、授業の流れを崩さず試せます。教員自身がICTに慣れる場にもなります。

### 全員が同じ使い方をする必要はない

クラス全員が同じアプリを同じ操作で使わなくてよいのがICTの特徴です。書くことが得意な子は書く、タイピングが楽な子は入力する、写真で記録したい子は撮る——**学び方の選択肢が増えること自体が、ICT活用の価値**のひとつです。

→ 教育DX全体の考え方は[教育DXとは何か？現場教員が押さえておきたい基礎知識](/articles/kyoiku-dx-kiso)もご参照ください。

## 2. 教師用端末1台から始めるICT活用

児童生徒の端末を使う前に、まず「教師が前に出して見せる」使い方が最も導入しやすいステップです。

### 大型提示装置への接続

タブレットをHDMIケーブル・ミラーリング・無線接続でテレビやプロジェクタにつなぎ、画面を拡大表示します。

**できること：**
- 教科書のページを拡大して全員に見せる
- 写真・動画・Webサイトを大きく提示する
- 板書の補助として図・グラフ・地図を表示する

### 手元の拡大提示

書画カメラ（実物投影機）の代わりとして、タブレットのカメラを使って実物・手元・ノートを拡大して見せることができます。

**活用例：**
- 工作・実験の手順を手元カメラで見せる
- 児童のノートの良い例をリアルタイムで共有する
- 小さな部品や標本を拡大して観察させる

### タイマーと予定の見える化

タイマーアプリや時計を大型提示に表示することで、**活動の見通し**を全員と共有できます。「あと5分」「次は発表」という情報を視覚化することは、特に見通しを持つことが難しい子どもにとっても有効です。

### ワークシート・板書の提示

事前に作成したワークシートや思考ツールをスライド形式で表示すると、配布の手間が省け、指示の説明もスムーズになります。

## 3. 児童生徒が使うICT活用の基本場面

教師用の活用に慣れたら、少しずつ児童生徒の端末活用へ広げます。以下の6場面から、授業の目的に合わせて選んでください。

### ① 調べる

Web検索・デジタル教科書・動画教材などを使って情報を収集します。

**ポイント：**
- 「調べればわかること」と「自分で考えること」のバランスを意識する
- 信頼できる情報源（文科省・自治体・公共機関など）を事前に示しておく
- 検索の仕方・情報の選び方も一緒に学ぶ機会になる

### ② 写真を撮る

観察・制作・活動の様子を写真や動画で記録します。

**活用例：**
- 理科の観察記録（植物の成長・実験の様子）
- 工作・図工作品の途中経過・完成写真
- 体育の技のフォームを動画で確認

**注意：** 写真・動画には個人が映り込むことがあります。撮影ルールと個人情報への配慮は事前に確認してください（後述）。

### ③ 考えを入力する

テキスト・付箋ツール・思考マップアプリなどで自分の考えを入力します。

**メリット：**
- 書くことが苦手な子も、音声入力・フリック入力で記録できる
- 修正・並び替えが容易で、思考の整理がしやすい
- 手書きより読みやすく、共有に向いている

### ④ 発表する

スライド・写真・デジタルポスターなどを使って考えを発表します。

**ポイント：**
- スライド作成が目的にならないよう、内容の質を重視する
- 1人1台であれば、全員が画面を持ちながら発表できる
- 発表が苦手な子が「画面を見せながら話す」ことで参加しやすくなる場合がある

### ⑤ 共有する

クラウドやアプリを使って、グループ・クラス全体で考えを共有します。

**活用例：**
- 共有ノート・スプレッドシートに意見を書き込む
- 発表スライドをクラスルームで共有し、互いに閲覧する
- 付箋ツールにグループ全員が意見を貼り付けて整理する

### ⑥ 振り返る

学習の最後に今日の学びや感想を入力・記録します。

**メリット：**
- 蓄積すると成長の記録になる
- 教員が全員分の振り返りを素早く確認できる
- 次時の授業設計に活かしやすい

## 4. 教科別の簡単な活用例

教科ごとに「この場面でこう使える」という具体例を整理します。端末の種類に関わらず活用できる方法を中心に紹介します。

### 国語

| 場面 | ICT活用例 |
|---|---|
| 音読 | タブレットで自分の音読を録音し、聴き直す |
| 作文・日記 | キーボードや音声入力でメモを作り、後で整理する |
| 語句・慣用句調べ | Web辞書・デジタル国語辞典で意味を調べる |
| 物語の読み取り | 重要場面の写真や動画で作品の背景を調べる |

### 算数・数学

| 場面 | ICT活用例 |
|---|---|
| 図形 | デジタル操作で図形を回転・移動させて理解する |
| 考え方の共有 | 解き方の写真を全員で見て比較する |
| 途中式の確認 | ノートの途中式を撮影してグループで見せ合う |
| グラフ | デジタルツールでグラフを作成・比較する |

### 理科

| 場面 | ICT活用例 |
|---|---|
| 観察記録 | 生き物・植物の変化を写真で時系列に記録する |
| 実験 | 実験の様子を動画で確認し、変化を比較する |
| 資料 | 細胞・天体・地層など、実物が見せられない対象を動画・画像で提示する |
| 考察 | 観察結果を入力して共有し、クラスで比較する |

### 社会

| 場面 | ICT活用例 |
|---|---|
| 地図 | デジタル地図で場所・距離・地形を確認する |
| 資料 | 歴史写真・統計グラフ・新聞記事を大型提示する |
| 調べ学習 | 都道府県・産業・歴史を分担して調べてまとめる |
| まとめ | 調べた内容をスライドや新聞形式でまとめる |

### 生活・総合的な学習の時間

| 場面 | ICT活用例 |
|---|---|
| 活動記録 | 体験・校外学習の様子を写真・動画で記録する |
| 発表資料 | 調べたことをスライドや動画でまとめる |
| 振り返り | 活動の前後を比較し、成長を記録する |

### 体育・音楽・図工

| 場面 | ICT活用例 |
|---|---|
| 体育 | 技・フォームを動画撮影してフィードバックする |
| 音楽 | 演奏・歌声を録音して聴き直す |
| 図工 | 制作過程を写真で記録してポートフォリオにする |

## 5. ICT活用で授業が変わるポイント

単にデジタルになるだけでなく、ICTによって授業の質が変わる具体的な場面を紹介します。

### 見えにくいものが見える

顕微鏡の映像・高速度カメラ・衛星写真・過去と現在の比較——これらはICTなしでは教室で見せることが難しい情報です。「百聞は一見にしかず」を実現しやすくなります。

### 考えを「残す」ことができる

紙のワークシートと違い、デジタルの記録は蓄積・検索・比較ができます。学期初めと学期末の作文を見比べる、観察記録を並べて変化を確認する——こうした「振り返り」が具体的になります。

### 発表が苦手な子も表現しやすくなる

全員の前で話すことに緊張する子が、**画面を持ちながら小さな声で説明する・スライドに沿って読む**という形で参加しやすくなることがあります。ICTは発表の方法の選択肢を広げます。

### 教員が全員の考えを把握しやすくなる

紙では教室を回って一人ずつ見る必要がある子どもの考えが、共有ツールを使えば教師端末から全員分を一覧で確認できます。理解の程度・つまずきのパターンをリアルタイムに把握して、その場で授業を調整できます。

## 6. 失敗しやすいICT活用

ICTを導入する際によく起こる問題を事前に知っておくことで、避けやすくなります。

### アプリありきで授業を組む

「このアプリが面白そうだから使いたい」という出発点は、教育目標がぼやけやすくなります。**「この授業で何を身につけさせるか」が先**で、そのためにどのICTが有効かを選ぶ順番が大切です。

### 操作説明だけで時間が終わる

慣れていないアプリを初めて使う授業は、操作の説明・トラブル対応で授業時間の多くが費やされることがあります。**ICT操作は別の時間で習得させ、授業では内容に集中させる**のが効率的です。

### 全員に同じ速度を求める

操作の得手不得手はクラス内で大きく異なります。「全員が同じ操作を同じ時間で終える」を前提にした授業設計は、早い子が暇になり、遅い子がついていけない状況を生みます。**終わった子は次の活動に進む・早い子が遅い子を助ける**などの設計を工夫しましょう。

### 端末トラブル時の代替案がない

充電切れ・接続不良・アプリが起動しない——こうしたトラブルはある程度起こります。**「端末が使えなくてもこの部分はできる」という授業の骨格**を持っておくと、慌てずに対応できます。

### 情報モラルや個人情報への配慮が不足する

ICT活用を広げていく中で、情報の扱い方を丁寧に指導する機会は必ず設けてください（後述）。

## 7. 安全に使うためのポイント

### 写真・動画の扱い

児童生徒が映っている写真・動画は個人情報です。次の点を事前に確認・指導してください。

- **撮影してよいもの・いけないもの**のルールをクラスで共有する
- 他の人の顔が映った写真をSNSに投稿しない
- 学校端末で撮影した写真は個人のクラウドへ勝手に保存しない
- 保護者の同意が必要な使い方かどうかを確認する

### アカウント管理

- 学校配布のアカウントと個人アカウントを明確に分ける
- パスワードを他者に教えない・画面を見せない
- 使い終わったらサインアウトする（共用端末の場合）

### 校内ルールの確認

使用できるアプリ・サービス・クラウドストレージは学校・教育委員会によって制限されている場合があります。新しいサービスを使う前に**ICT担当・管理職に確認**してください。

### 持ち帰りと家庭連携

端末の持ち帰りを認めている学校では、家庭での使用ルール・家庭でのWi-Fi接続の可否・保護者への説明が必要です。

### 生成AIの利用について

ChatGPTなどの生成AIは、本記事のICT活用とは別のカテゴリで扱います。授業中に生成AIを使わせる際は、**個人情報を入力させない・生成物をそのまま提出させない**などの指導と、校内ルールの確認が必要です。

→ 詳しくは[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)をご参照ください。

## 8. 特別支援教育にもつながる視点

本記事の内容の多くは、通常学級・特別支援学級・特別支援学校のどちらでも活用できます。特別支援教育の文脈では、次の4つの視点が特に重要です。

### 見通しを持つ

タイマー・スケジュール表示・手順の提示は、次に何が起こるかを知ることで安心して活動できる子どもにとって特に有効です。

### 表現方法を増やす

話す・書く以外に、タップする・写真で示す・録音した音声を使うという選択肢を持つことで、自分の考えを伝えやすくなる子どもがいます。

### 読み書きの負担を調整する

音声入力・読み上げ機能・文字サイズの変更は、読み書きにかかる時間や労力を調整する手段です。「特別な対応」ではなく、クラス全体が使える設定として整えると、必要な子が自然に使えます。

### コミュニケーションを補助する

AACアプリ・絵カードアプリ・選択肢提示などは、話し言葉だけに頼らないコミュニケーションの方法です。

→ 特別支援教育でのICT活用の詳細は[特別支援教育でICTを活用する：タブレット・AAC・読み上げソフトの選び方](/articles/tokubetsu-shien-ict)をご参照ください。

## まとめ

タブレットを使った授業ICT活用は、**大きく始めなくてよい**です。

**まず試せる4つのステップ：**

1. **提示** — 教師用端末を大型テレビにつなぎ、写真・動画・資料を見せる
2. **記録** — 子どもたちが観察・活動・制作を写真・メモで記録する
3. **共有** — 考えをデジタルでグループ・クラスに共有する
4. **振り返り** — 学習の最後に今日の学びを入力・保存する

**守ってほしい3原則：**

1. ICTは手段であり、目的は子どもの学び
2. 全員が同じ使い方をする必要はない
3. 写真・個人情報・アカウントは事前にルールを決めてから使う

タブレットは授業を派手にする道具ではありません。「見る・調べる・考える・表す・共有する・振り返る」という学びのプロセスを、その子にとってより自然にする道具です。

まずは1場面・1教科から、小さな一歩を踏み出してみましょう。
