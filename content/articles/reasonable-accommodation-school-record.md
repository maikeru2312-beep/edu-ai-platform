## はじめに

「合理的配慮」という言葉は、特別支援教育の文脈だけでなく、通常学級・進学先・試験・就労の場面でも広く使われるようになっています。一方で、学校現場では「どこまで対応するのか」「何をどう記録すればよいのか」「保護者からの要望をどう整理するか」で迷う場面が少なくありません。

本記事では、合理的配慮の基本的な考え方から、相談・合意形成・記録・見直しの流れ、ICTとの関係、個人情報の注意点まで、学校現場で使いやすい形で整理します。

> 合理的配慮の具体的な判断・対応は、学校・設置者・教育委員会の方針によって異なります。本記事は一般的な考え方の整理であり、個別ケースの判断を示すものではありません。**所属校・特別支援コーディネーター・管理職と連携の上**進めてください。

---

## 1. 合理的配慮とは何か

合理的配慮とは、**障害のある人が他の人と平等に社会参加・教育を受けられるよう、その人の必要に応じて行う適切な変更・調整**のことです（障害者差別解消法）。

学校教育の文脈では、「障害のある児童生徒が他の児童生徒と同じように学ぶために必要な調整を、学校として無理のない範囲で行うこと」と理解できます。

### 「特別扱い」ではなく「学びへのアクセス保障」

合理的配慮はよく「特別扱いでは?」と誤解されます。しかし、考え方の出発点は逆です。

- 配慮**なしの状態**では、その子どもは学習内容にアクセスできていない
- 配慮によって、**他の子どもと同じスタートラインに立てる**
- 目標・評価の基準を変えるのではなく、**その子どもが目標に向かえる手段を調整する**

「みんなと同じにする」のではなく「その子がその子らしく学べる」ようにするための調整が、合理的配慮の本質です。

## 2. 合理的配慮と基礎的環境整備の違い

合理的配慮を理解する上で、**基礎的環境整備との違い**を整理しておくことが重要です。

| | 基礎的環境整備 | 合理的配慮 |
|---|---|---|
| **対象** | 全員・学校全体 | 特定の児童生徒 |
| **実施主体** | 学校・自治体・国 | 学校（本人・保護者との合意を経て） |
| **内容** | 施設・設備・制度・体制 | 個別の調整・変更 |
| **例（ICT）** | 1人1台端末の整備、大型提示装置の設置 | 特定の児童に音声入力・拡大表示を認める |

**具体例で整理すると：**

- 教室に大型テレビがある → **基礎的環境整備**
- 視覚に困難さのある児童の座席位置を前方に調整する → **合理的配慮**
- 全員がタブレットで板書を撮影できるルールにする → **基礎的環境整備**
- 書字に時間がかかる特定の児童に、授業後に板書写真を確認することを認める → **合理的配慮**

基礎的環境整備が整っていることが、個別の合理的配慮を実施しやすくする土台になります。

→ 学校全体でのICT環境整備については[タブレット1台で始める授業ICT活用【GIGAスクール対応】](/articles/tablet-ict-jugyo-giga)もご参照ください。

## 3. 合理的配慮は誰が申請するのか

合理的配慮のプロセスは、多くの場合**本人または保護者からの意思表明・相談**を出発点とします。ただし、それだけが始まりではありません。

**意思表明のパターン：**

- 保護者から「〇〇が難しいので何か対応してほしい」と相談を受ける
- 本人が「〇〇はつらい」と教員に伝える
- 教員が授業中の困難さに気づき、保護者に相談を提案する
- 前の学校・学年から引き継ぎ情報がある

**重要なのは「建設的対話」**

合理的配慮は、保護者・本人の要望に対して学校が一方的に「認める・認めない」を決めるものではありません。**本人・保護者・学校が対話を通じて、必要な調整と実現可能な方法を一緒に検討するプロセス**です。

また、「申請書がなければ始められない」と考えすぎず、まず**相談の記録を残すことから始める**ことで、対話の証跡と校内共有の基盤が作れます。

## 4. 学校現場での進め方

合理的配慮の実践は、次の流れで進めることが基本です。

### Step 1：相談を受ける・困難さを確認する

相談を受けたら、まず**「どの場面でどんなことが難しいか」を具体的に確認**します。「勉強が大変」という訴えの背景に何があるかを丁寧に聞くことが出発点です。

確認すべき観点：
- いつ（どの授業・時間帯・場面で）困っているか
- 何が（書く・読む・聞く・集中・コミュニケーションなど）難しいか
- 今どんな工夫をしているか、効果はあるか
- 本人はどう感じているか

### Step 2：すでに行っている支援を確認する

学校がすでに行っている配慮・工夫と、その効果を整理します。「今やっていることで十分か」「何が足りないか」の起点になります。

### Step 3：調整案を検討する

困難さと現状の支援を踏まえ、**可能な調整案を複数出します**。一つに絞るのではなく、「A案・B案・代替案」の形で選択肢を持つことで、本人・保護者との対話がしやすくなります。

### Step 4：本人・保護者と合意形成する

提案した調整案について、本人・保護者と話し合います。**「何が目的か」「どう評価するか」「いつ見直すか」**を含めて確認し、合意内容を記録します。

### Step 5：校内で共有する

合意した配慮内容を、関係する教員・スタッフに共有します。担任だけが知っている状態では、交替・複数教科・校外活動で支援が途切れます。**共有範囲と情報管理の方法**も確認してください。

### Step 6：実施・見直す

一定期間実施した後、**「配慮の効果はあるか」「内容を変える必要があるか」**を本人・保護者と確認します。合理的配慮は固定ではなく、子どもの実態の変化に応じて更新するものです。

## 5. 記録に残すべき項目

記録は「何をしたか」だけでなく、「なぜ必要か」「いつ見直すか」まで残すことで、校内共有・引き継ぎ・見直しに役立ちます。

**記録の基本項目：**

| 項目 | 内容例 |
|---|---|
| 相談日 | 初回相談日・合意日 |
| 相談者 | 保護者氏名（記録上のみ）、担任、特支コーディネーター |
| 本人・保護者の願い | 「書くことへの苦手さを軽減してほしい」 |
| 困難さが生じている場面 | 「国語・理科の板書を書き写す場面で特に時間がかかる」 |
| 学校側の見立て | 「書字速度が遅く、学習内容の理解に使える時間が不足している」 |
| 検討した調整案 | A案：板書写真の確認を認める、B案：要点プリントの配布、C案：タブレットへの入力 |
| 実施する配慮内容 | 「授業後に板書写真を確認できるようにする（A案）」 |
| 実施期間 | 「2026年9月〜12月（1学期間）」 |
| 担当者 | 担任・特支コーディネーター |
| 校内共有範囲 | 「学年団・管理職のみ」 |
| 見直し時期 | 「2026年12月の個別面談時」 |
| 実施後の様子 | （見直し時に記載） |

## 6. 個別の教育支援計画・個別の指導計画との関係

合理的配慮は、**個別の教育支援計画（IEP）・個別の指導計画とセットで管理する**ことが望ましいです。

### 個別の教育支援計画（個別支援計画）

学校・家庭・関係機関が連携して作成する長期的な支援の計画です。合理的配慮の内容も、必要に応じてこの計画に記載します。

**記載に向く内容：**
- 中・長期的に継続する配慮
- 関係機関（医療・福祉・相談支援機関など）との連携内容
- 学年をまたいで引き継ぐ事項

### 個別の指導計画

授業・学習場面での具体的な支援目標・手立てを記載する計画です。

**記載に向く内容：**
- 各教科の授業での配慮（座席・教材・評価方法）
- 学習上の短期目標と手立て
- 効果の確認・見直し時期

### 実用上のポイント

書類が複雑になりすぎると、実際に使われない記録になってしまいます。**「作って終わり」ではなく、教員が実際に参照・更新できる粒度**を意識してください。校内でのフォーマット統一・共有フォルダの整備も合わせて検討してみてください。

## 7. よくある合理的配慮の例

以下は学校現場でよく検討される配慮の例です。必要性・実施可能性は個別の状況によって異なります。

**学習場面での配慮例：**

- 座席位置の調整（前方・端・壁側など、刺激の少ない位置）
- 板書・配布資料の拡大・フォント変更
- 読み上げ機能・音声教材の使用
- 書字量・筆記量の調整（口頭回答・タイピング・記録量の調整）
- タブレットや音声入力・拡大表示などICT機器の活用
- 課題の提示方法の工夫（文字より図・写真・視覚的提示）
- 発表方法の選択肢を広げる（口頭・スライド・記録での提示）

**評価・試験場面での配慮例：**

- 別室での受験
- 試験時間の延長
- 問題の読み上げ・拡大印刷
- 解答方法の変更（記述→選択、手書き→タイピング）

**環境・生活面での配慮例：**

- 感覚過敏への対応（照明・音・においの調整、イヤーマフの使用）
- 活動の見通しを視覚的に示す（スケジュール表・タイマー）
- 不安が高まったときの休憩場所の確保

→ ICTを活用した配慮の具体例については[特別支援教育でICTを活用する：タブレット・AAC・読み上げソフトの選び方](/articles/tokubetsu-shien-ict)をご参照ください。

## 8. どこまで対応すべきか

合理的配慮は「すべての要望を受け入れる」ものではなく、**必要性・実現可能性・代替案**を含めて検討します。判断の観点を整理します。

**判断の観点：**

| 観点 | 確認内容 |
|---|---|
| 必要性 | その配慮は、その子どもが学びにアクセスするために本当に必要か |
| 本質変更の有無 | 学習目標・評価の本質を変えてしまわないか |
| 他の児童への影響 | 他の児童生徒の学習権を大きく妨げないか |
| 継続可能性 | 学校体制として無理なく続けられるか（人員・費用・時間） |
| 安全性 | 安全面に問題がないか |
| 代替手段 | この方法でなければできないか、代わりの方法はないか |

**「できない」で終わらない**

検討の結果、要望通りの対応が難しい場合でも、**「この部分なら対応できる」「こういう代替案はどうか」という建設的な代替提案**を必ずセットで伝えることが大切です。一方的に断ることは、建設的対話の趣旨から外れます。

## 9. 記録の文例：よい書き方・避けたい書き方

合理的配慮の記録は、**「なぜ必要か」を具体的に書くこと**が、引き継ぎや見直しでの精度を高めます。

### 避けたい書き方

> 「○○が苦手なので配慮する」

何が苦手なのか、どの場面で、どんな配慮をするのかが不明確で、引き継ぎに使えません。

### よい書き方の例

**板書・記録の困難さへの対応：**
> 「板書を書き写す速度が遅く、書くことに集中することで授業内容の理解に使える時間が不足している。そのため、授業後に板書写真または要点プリントを確認できるようにする。」

**集中・環境への対応：**
> 「周囲の音や動きにより課題への取り組みが中断しやすい。そのため、必要に応じて刺激の少ない座席位置・場所で課題に取り組む時間を設定する。」

**評価方法の調整：**
> 「学習目標は維持しつつ、手書きによる記述が困難なため、タブレットへの入力または口頭での説明に代えて評価する。」

「○○が苦手」という表現よりも、**「どの場面でどんな困難さが生じ、そのために何をするのか」**という構造で書くと、実用的な記録になります。

## 10. ICT・アクセシビリティとの関係

ICTは合理的配慮を実現する有効な手段のひとつですが、**「端末を渡せば配慮になる」ではありません**。

**ICTが配慮として機能するためには：**

1. **何の困難さに対して** 使うのかを明確にする
2. **どの機能を** 使うのかを具体的に決める
3. **どの場面で** 使うのか（全教科か、特定の教科か）を確認する
4. **操作の習得** を支援する（機能があっても使い方がわからなければ意味がない）
5. **効果の確認** を定期的に行う

**ICTを活用した配慮の例：**

| 困難さ | ICTによる対応例 |
|---|---|
| 板書を書き写すことが難しい | タブレットで板書を撮影・記録する |
| 文字を読むことに時間がかかる | 読み上げ機能・音声教材で聴く |
| 手書きが難しい | タブレット入力・音声入力で記録する |
| 次に何があるかが不安 | スケジュールアプリ・タイマーで見通しを持つ |
| 口頭での意思表出が難しい | AACアプリ・絵カードアプリで伝える |

→ 特別支援教育でのICT活用の詳細は[特別支援教育でICTを活用する：タブレット・AAC・読み上げソフトの選び方](/articles/tokubetsu-shien-ict)をご参照ください。

## 11. 個人情報と校内共有の注意点

合理的配慮の記録には、**障害・困難さ・家庭との相談内容など、慎重に扱うべき情報が含まれます**。

### 共有範囲を明確にする

「誰に・何を・どこまで共有するか」を本人・保護者と確認してから、校内共有を行います。「全職員に共有してよいか」「学年団のみでよいか」「非常勤講師や部活動顧問にも伝えるか」など、範囲を丁寧に確認してください。

### 保管・アクセスの管理

合理的配慮の記録は、個人情報を含む文書です。校内の保管ルール（紙・電子ファイルの保管場所・アクセス権限）を確認してください。

### デジタルツール・AIへの入力に注意する

合理的配慮の記録や相談内容を、メールや外部クラウドサービス・生成AIに入力する際は特に注意が必要です。氏名・診断名・家庭状況などの要配慮個人情報が含まれる内容は、**校内が許可した安全な環境のみで扱い、外部サービスには入力しない**ことを基本としてください。

→ 個人情報と生成AIに関する詳細は[学校で生成AIを使うときの個人情報・セキュリティの注意点](/articles/school-generative-ai-privacy-security)をご参照ください。

## 12. よくあるQ&A

**Q. 口頭で相談されただけでも合理的配慮は始められるか?**

A. 口頭相談から配慮を始めることは可能ですが、後から「言った・言わない」になるリスクを避けるためにも、相談の記録は必ず残すことを強くお勧めします。記録することが信頼関係を守ることにもつながります。

**Q. 診断書がないと合理的配慮はできないのか?**

A. 合理的配慮は、必ずしも医師の診断書が必要なわけではありません。学習・生活上の困難さの実態に基づいて検討することが基本です。ただし、診断・専門機関の情報がある場合は、適切な配慮を検討する上での参考になります。具体的な判断は学校・設置者の方針によります。

**Q. 保護者の希望をすべて受け入れる必要があるのか?**

A. すべての要望を受け入れる義務はありませんが、建設的な対話と代替案の提示が求められます。「過重な負担」に当たると判断した場合は、その理由と代替案を丁寧に説明することが重要です。

**Q. 他の児童生徒から「不公平」と言われたらどうするか?**

A. 情報モラルを含む形で「人によって必要なサポートは違う」という考え方を、学級・学年全体に伝える機会になり得ます。個別の情報（誰がどんな配慮を受けているか）は伝えませんが、「違いを認め合う学級づくり」の文脈で対話することができます。

**Q. 通常学級でも合理的配慮は必要か?**

A. 必要です。合理的配慮は特別支援学校・特別支援学級だけでなく、通常学級在籍の児童生徒にも適用されます。通常学級担任も、合理的配慮の基本を理解しておくことが求められています。

**Q. 一度決めた配慮はずっと続ける必要があるのか?**

A. ありません。合理的配慮は子どもの実態の変化に応じて見直すものです。「不要になった配慮」も、「効果が確認できなかった配慮」も、本人・保護者と確認の上で変更・終了することができます。

**Q. 入試や定期テストではどう考えるか?**

A. 学校内の定期テストでは、合意に基づいた配慮（時間延長・別室受験・問題の拡大等）が実施されるケースがあります。高校入試・資格検定等では、各実施機関が独自の申請フロー・対応を定めています。入試に向けた配慮申請は、早期から準備・相談を始めることが重要です。

合理的配慮は「特別扱い」ではなく、**その子どもが学びにアクセスできるための調整**です。

**実践のポイント：**

1. **建設的対話** — 本人・保護者・学校で対話しながら合意形成する
2. **具体的な記録** — 「なぜ必要か」「何をするか」「いつ見直すか」を残す
3. **校内共有** — 担任一人で抱えず、関係者で情報を共有する
4. **定期的な見直し** — 固定せず、子どもの実態の変化に合わせて更新する
5. **ICTの活用** — 手段として有効に使い、使うこと自体を目的にしない

「どこまで対応するか」に迷ったときは、**「この子どもが学びに参加するために必要な最低限の調整か」「代替手段はないか」**という視点に立ち返ってください。
