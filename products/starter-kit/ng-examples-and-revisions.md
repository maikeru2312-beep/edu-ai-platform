# NG例とその修正版（スターターキット同梱）

状態: 初稿 ／ 事例はすべて架空

> 「やってはいけない使い方」と「同じ目的を安全に達成する使い方」を対で示す。線引きを体で覚えるための教材。

---

## 1. 所見をAIに丸投げする

❌ **NG**
```
児童Aは自閉症で、こだわりが強く…（実名・障害名・具体的な様子を入力）
この子の通知表所見を書いて。
```
問題：個人情報・要配慮個人情報の入力／子どもの実態の判断をAIにさせている。

✅ **修正版**
```
私が書いた所見の下書きを、保護者に伝わる前向きな文体に整えてください。事実は変えないこと。
下書き：「ある児童。係活動に最後まで取り組み、友達に声をかける姿が見られた。」
```
ポイント：固有名は伏字、判断と事実は自分。AIは整文だけ。

---

## 2. 支援方針をAIに決めさせる

❌ **NG**
```
このクラスの○○さんに、どんな合理的配慮をすべきか決めて。
```
問題：配慮の決定は本人・保護者・関係機関と連携して人が行うもの。AIに決めさせない。

✅ **修正版**
```
読みに困りごとがある場合の一般的な支援の選択肢を、利点と留意点つきで挙げてください。特定の子の判断はしないでください。
```
ポイント：あくまで一般的な選択肢。決定は人が行う。

---

## 3. 保護者向け文書をそのまま配布

❌ **NG**
- AIが出した学級通信をそのまま印刷・配布。

問題：事実誤りや不適切表現が残るリスク。公式発信の責任は教員。

✅ **修正版**
- AIは「箇条書き→文章」「言い換え」まで。**シート8（AI出力確認欄）**を通し、固有の出来事・配慮は自分の言葉で加えてから配布。

---

## 4. 個人情報を入れて会議メモを整理

❌ **NG**
```
（実名・家庭状況・診断を含む会議メモを貼り付け）これを整理して。
```
問題：要配慮個人情報の入力。

✅ **修正版**
```
次のメモを、決定事項・宿題・次回までにやることに整理してください。
メモ：「（伏字・一般化した内容）」
```
ポイント：貼る前に伏字・一般化。機微情報は入れない。

---

## 5. AIの答えを事実確認せず使う

❌ **NG**
- AIが書いた「制度の説明」「日時」をそのまま資料に転記。

問題：もっともらしい誤り（事実と違う内容）が混ざることがある。

✅ **修正版**
- 数値・日時・制度・名称は**一次情報で裏取り**してから使う。シート8の項目1で確認。

---

## まとめ：3つの合言葉

1. **固有名は伏字、判断は自分。**
2. **AIは下書き、仕上げは人。**
3. **使ったら必ず確認欄（シート8）を通す。**

> すべての例は架空。実在の児童・学校・自治体を想起させる記述は含めない。
