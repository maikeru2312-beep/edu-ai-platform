# コンプライアンスチェック（COMPLIANCE CHECK）

最終更新: 2026-06-28（初稿 / QA Agent 相当）

> 本書は商品群の**販売前リスク確認**をまとめたもの。チェックが全て埋まるまで公開しない。法的・制度的な最終判断はユーザー本人（および必要に応じて所属先・専門家）が行う。

---

## 1. 公務員の服務・兼職兼業（最重要）

公立学校教員は、地方公務員法等により**営利目的の兼業・副業に制限**がある。デジタル商品の有料販売は、職務専念義務・信用失墜行為の禁止・兼業許可の観点に関わりうる。

- [ ] 所属自治体・教育委員会の**兼職兼業（営利企業従事制限）の規程**を本人が確認した
- [ ] 有料販売に**許可・届出が必要か**を確認した（必要なら手続き前に販売しない）
- [ ] 学校名・職名・勤務校が**特定されない形**で販売・発信する方針を確認した
- [ ] 私的時間・私的環境（個人PC・個人アカウント）で制作した前提を維持している

> ⚠️ 本商品群は「販売前確認が必要」であることを各販売ページ案・運用メモに明記する。確認が済むまで**有料販売は行わない**。

---

## 2. 個人情報・要配慮個人情報

- [ ] 実在する児童生徒・保護者・同僚・勤務校が**推測できる記述がない**
- [ ] 記入例・事例は**すべて架空**（氏名・地名・学校名は実在しない一般名・伏字）
- [ ] プロンプトやテンプレートが、利用者に**個人情報の入力を促す設計になっていない**
  - 各プロンプトに「入力してはいけない情報」を明記
  - 各シートの記入例で「氏名は伏字・障害名は書かない」等を例示
- [ ] 児童生徒の実態を**AIに断定させる**表現がない（「AIに見立てさせる」「診断させる」等を含めない）

---

## 3. 著作権

- [ ] 文科省資料・他サイト・書籍の**長文転載がない**
- [ ] 引用する場合も最小限・出典明示・自分の言葉での再構成を基本とする
- [ ] 教育DXナビ既存記事は**コピペでなく商品用に再構成**している
- [ ] 図表・画像は自作または権利クリアなもののみ（現時点はテキスト主体）

---

## 4. 表現（誇大・断定の禁止）

- [ ] 「必ず売れる」「必ず収益化できる」等の**収益保証表現がない**
- [ ] 「残業が必ず減る」「成績が上がる」等の**効果の断定がない**（「〜に役立つ場合がある」等に統一）
- [ ] 「AIに子どもの理解を丸投げする」ことを**推奨する表現がない**（むしろ戒める主張を入れる）
- [ ] 特定サービス・企業を不当に推奨／中傷していない

---

## 5. 教育的・倫理的妥当性

- [ ] 「生成物は必ず教員が確認する」原則が全商品に通底している
- [ ] 特別支援教育で「AIに任せてはいけないこと」を明示している（商品C・D）
- [ ] 合理的配慮・個別の指導計画など制度に関わる記述は「所属校・教育委員会の方針に従う」前提を併記
- [ ] 医療・診断・法的助言にあたる断定をしていない

---

## 6. 技術・サイト保護（AdSense審査保護）

- [ ] **本番サイトのコードを触っていない**（products/ 配下の新規追加のみ）
- [ ] AdSense コード（`ca-pub-3801092904087307`）に変更なし
- [ ] GA4（`G-XCORRZCZM5`）に変更なし
- [ ] Search Console verification に変更なし
- [ ] robots / sitemap / canonical に変更なし
- [ ] 販売ページを本番サイトに公開していない／自動投稿していない

---

## 7. 個別指摘（今回の原稿で特に注意した点）

| 箇所 | リスク | 対応 |
|---|---|---|
| プロンプト集（B・D） | 利用者が氏名・成績を入力する恐れ | 各プロンプトに「入力NG情報」「出力後の確認」を必須付記 |
| 特別支援テンプレ（C） | 実在児童を想起させる記入例 | 記入例は架空・伏字・抽象化。障害名の記入を求めない欄設計 |
| スターターキット主張（D） | 「AIで支援を効率化」が独り歩き | 「子どもの理解をAIに丸投げしない」章を冒頭近くに配置 |
| 販売コピー（launch/） | 効果・収益の誇大化 | 断定回避・体験談ねつ造なし・「個人差」「校内ルール優先」を明記 |
| 全体 | 公務員の兼業抵触 | 各販売ページ案に「販売前に服務確認」の注記、本書§1で一元管理 |

---

## 8. 販売前に人間（ユーザー）が必ず確認すべきこと（要約）

1. **服務・兼職兼業の許可要否**（§1）— 最優先。
2. 全原稿の最終目視（架空事例の徹底）。
3. 価格・特商法表記・プラットフォーム規約。
4. 無料/有料の配布手段と個人情報の取り扱い方針。
5. 既存サイトへリンクを足す場合の範囲（最小限・別承認）。
