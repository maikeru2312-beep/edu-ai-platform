# シート4：合理的配慮とICT活用 整理シート

> 同じタブレットでも、目的が違えば位置づけが変わる。「合理的配慮」と「教育課程上の指導・支援」を分けて整理する。

| 問い | 記入欄 |
|---|---|
| ① 何に困っているか（困りごと・本人の願い） | |
| ② そのICTで何ができるようになるか | |
| ③ これは「合理的配慮」か？（参加・アクセスの保障） | □ はい／□ いいえ／□ 両方 |
| ④ これは「指導・支援」か？（力を育てる目的） | □ はい／□ いいえ／□ 両方 |
| ⑤ 本人・保護者と合意できているか | □ 済／□ これから |
| ⑥ どこに記録するか（計画・記録の種類） | |
| ⑦ 見直しの時期・観点 | |

> ③④は排他ではない。**同じ手段が配慮と指導の両面を持つ**こともある。大切なのは「何のために使うか」を言葉にすること。

---

### 記入例（架空）

- 場面：テストでの読みの困りごと／使うICT：問題文の読み上げ。
- ③合理的配慮：はい（読みの負担を取り除き、内容理解を公平に評価するため）。
- ④指導：いいえ（読みの力を鍛える目的ではない）。
- ⑥記録：個別の教育支援計画・テスト時の配慮として記録（→シート5）。
- ⑦見直し：学期末に、本人の負担感と理解度で見直す。

> ⚠️ 配慮の決定は本人・保護者・関係機関と連携し、所属校・教育委員会の方針に従う。本シートは整理の下書き。
