# シート8：AI出力をそのまま使わないための確認欄

特別支援教育×校務 ／ 印刷してA4で使用 ／ 記入例は架空

> 生成AIの出力は、必ずこの欄を通してから使う。「もっともらしい誤り」を見逃さないための最終チェック。

**用途：**＿＿＿＿＿＿＿＿　**使ったツール：**＿＿＿＿＿　**日付：**＿＿＿

| # | 確認項目 | OK | 直す | メモ |
|---|---|:--:|:--:|---|
| 1 | 事実（日時・数値・名称）が正しい | □ | □ | |
| 2 | 個人情報・特定の子を想起させる記述がない | □ | □ | |
| 3 | 子どもの実態の断定・評価をAIにさせていない | □ | □ | |
| 4 | 表現が不適切・差別的・誤解を招くものでない | □ | □ | |
| 5 | 学校の公式発信として責任を持てる内容か | □ | □ | |

| 8 | 校内ルール（確認・配布手順）に沿っている | □ | □ | |

**最終判断：** □ このまま使う　□ 修正して使う　□ 使わない

### 記入例（架空）

- 用途：行事のお知らせ文の下書き。
- 項目1：日時が実際と1時間ずれていた→修正。
- 項目3：子どもの様子の一般表現に直し、特定の子の評価は削除。
- 最終判断：修正して使う。

> このシートを通すこと自体が「AIに丸投げしない」習慣になる。迷ったら管理職に確認。
