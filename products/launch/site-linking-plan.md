# 教育DXナビ からの導線案（site linking plan）

状態: 初稿（実装は別途ユーザー承認・最小限） ／ コード・SEO設定は触らない

> ⚠️ 本案は「どの記事から無料PDFへ自然に誘導できるか」の**設計メモ**。実際のリンク追加は、AdSense審査への影響を避けるため、ユーザー承認のうえ最小限・本文の自然な位置にとどめる。**canonical / sitemap / robots / GA4 / AdSense コードは一切変更しない。**

---

## 1. 導線の基本方針

- 商品の押し売りではなく「次に困りそうなことへの自然な手当て」として無料PDF(A)へ誘導。
- 入口は**無料PDF一択**に統一（有料は無料PDF内・note内から案内）。記事から直接有料へ送らない。
- 既存記事の本文・構成は極力変えない。追加するなら記事末の「関連・次の一歩」位置に1リンク。

## 3. 段階導線（記事 → 無料 → 有料）

```
既存記事（関連の1リンク）
   └→ 無料PDF配布ページ（note無料記事 or 自サイト固定ページ）
        ├→ 有料note『プロンプト30選』
        └→ 有料PDF『特別支援テンプレ集』→ スターターキット
```

---

## 4. 実装時の注意（承認後）

- [ ] 変更は本文への**関連リンク追加のみ**。メタ情報・構造化データ・canonical等は触らない。
- [ ] 1記事あたりリンクは1〜2本まで（過剰リンク・誘導の連発をしない）。
- [ ] 文言は断定・収益保証を避ける（「準備中」「役立つ場合があります」）。
- [ ] 追加後に sitemap / robots / canonical / GA4 / AdSense に影響がないことを確認。
- [ ] 公開前に差分をユーザーがレビュー。

---

## 5. やらないこと

- 本番サイトへの販売ページ公開。
- 全記事への一括リンク挿入（スパム的導線）。
- SEO設定・解析・広告コードの変更。
